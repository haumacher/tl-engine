Syntax

	$dict.values()

Beschreibung

Liefert die Werteliste eines Wörterbuchs. Die Liste kann, anders als bei Schlüsselmengen, doppelte Werte enthalten, da derselbe Wert mehreren Schlüsseln zugeordnet sein kann.

Parameter

Name Typ Beschreibung Pflicht Default
dict Fachobjekt Ein Wörterbuch, dessen Werte ausgegeben werden sollen. ja

Rückgabewert

Typ: Menge

Eine Liste, die alle Werte der Wörterbuches dict beinhaltet.

Beispiele

Einfaches Wörterbuch

	{
  dict = {
    "all": "alle",
    "some": "manche",
    "none": "keine"
  };

  $dict.values();
}

Ausgabe: Eine Liste mit den Werten ["alle", "manche", "keine"].
