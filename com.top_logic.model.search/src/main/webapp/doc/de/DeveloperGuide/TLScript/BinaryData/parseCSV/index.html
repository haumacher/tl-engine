Syntax

	$input.parseCSV()

Beschreibung

Interpretiert die Eingabe als CSV formattierten Text.

​Parameter

Name Typ Beschreibung Pflicht Default
input Binärwert Text im CSV-Format ja -

Rückgabewert

Typ: Liste

Erzeugt pro Zeile in der Eingabe eine Liste und fasst anschließend diese Listen in einer Liste zusammen.

Beispiel

{
    toParse = binary(
        name: "test.csv",
        data: "date; name; number
        15.07.2024; test; 5",
        contentType: "text/csv",
        encoding: "utf-8"
    );
    result = $toParse.parseCSV();
    $result[0]
}

Ausgabe: ["date", "name", "number"]

Der Aufruf von parseCSV() liefert die Liste [["date", "name", "number"], ["15.07.2024", "test", "5"]].
