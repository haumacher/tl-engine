Im Design-Modus erhalten Tableisten und auch die Side-Bar einen Menüknopf, mit dem diese Komponenten angepasst werden können.

Hier können neue Reiter hinzugefügt werden ("Tab hinzufügen") und die Reihenfolge der Tabs angepasst werden ("Tabs konfigurieren").

Tab hinzufügen

Über den Menüpunkt "Tab hinzufügen" kann in der jeweiligen Reiterleiste oder Side-Bar ein neuer Tab mit Label und optional einem Icon erstellt werden.

Ein neuer Tab auf oberster Ebene sollte immer mit einem Icon versehen werden. Da ein solcher Tab je nach Theme in der Sidebar angezeigt werden kann und diese im zusammengeklappten Zustand nur die Icons anzeigt, könnte ein Tab ohne Icon in dieser Situation nicht dargestellt werden.

Die gewünschte Position in der Tableiste oder Side-Bar kann danach über den Menüpunkt "Tabs konfigurieren" angepasst werden.

Tab-Inhalt füllen

Die neu erstellte Sicht ist noch leer. Sie kann anschließend mit Inhalt gefüllt werden. Um die Sicht mit einer neuen Komponente zu füllen, kann der Plus-Knopf in der Toolbar genutzt werden.

Für die neue zu erstellende Komponente kann aus einer Liste von Templates aufgewählt werden.

Die neue Sicht füllt dann den kompletten Bereich des neuen Tabs. Sollen mehrere Sichten nebeneinander in dem neuen Tab angezeigt werden, kann über den Menüpunkt "Layout bearbeiten" die Sicht in mehrere Bereiche aufgeteilt werden, siehe Aufteilung anpassen.

Unter-Tabs

Sollen unterhalb eines Tabs eine weitere Tab-Leiste angezeigt werden, so muss als neue Sicht die Option "Tabbar" gewählt werden. In dem sich öffnenden Dialog können dann Namen für die anzuzeigenen Unter-Tabs eingegeben werden. Die so neu erzeugte Tab-Leiste kann aber später noch genauso mit weiteren Tabs ergänzt werden.
