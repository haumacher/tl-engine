Syntax

	toString($value1, $value2, ...)

Beschreibung

Konvertiert und verkettet alle angegebenen Werte unter Verwendung des im System für den jeweiligen Typ registrierten Label-Providers. Mengenwerte werden automatisch geglättet.

Parameter

Name Typ Beschreibung Pflicht Default
values Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Die Werte, die verkettet werden sollen. ja

Rückgabewert

Typ: Zeichenkette

Eine Zeichenkette, die die Verkettung aller angegebenen values repräsentiert.

Beispiele

	toString("Greetings", 4, list("you", "!"))

Ausgabe: Greetings4you!

Dieses Ergebnis wird aus drei Teilen zusammengefügt. Der erste Teil ist die Zeichenkette Greetings, der zweite ist die Zahl 4 und der letzte Teil schließlich ist die Liste list("you", "!") bestehend aus zwei weiteren Zeichenketten. Alle diese Elemente werden zu der Ergbniszeichenkette zusammengefügt. Alle Listen werden dabei ausgepackt und nur ihre Elemente betrachtet.
