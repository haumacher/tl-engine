Syntax

	$type.attribute($name)

Beschreibung

Liefert das Attribut mit dem Namen name eines Typs type.

Das Attribut kann entweder ein Property mit primitivem Wertetyp sein, oder eine Referenz auf ein oder mehrere Objekte eines Fachobjekttyps.

Parameter

Name Typ Beschreibung Pflicht Default
type Zeichenkette Der Modelltyp, dessen Attribut ermittelt werden soll. ja
name Zeichenkette Name des gesuchten Attributs. ja

Rückgabewert

Typ: Fachobjekt

Das Attribut mit Angaben zum Namen, dem Definitons- und Zieltyp.

Beispiele

	`my.module:MyType`.attribute("myAttr")

Ausgabe: Das Attribut mit dem Namen "myAttr" sowie seinem Definitions- und Zieltyp.

Das zurückgegebene Attribut entspricht `my.module:MyType#myAttr`.
