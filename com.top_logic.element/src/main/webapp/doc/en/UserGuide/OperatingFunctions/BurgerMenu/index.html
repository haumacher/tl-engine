Spezeielle Funktionen einer GUI werden in TopLogic im sog. Burgermenü angezeigt. Der aufruf erfolgt durch klicken auf (1).

Im unteren Beispiel zeigt das aufgeklappte Burgermenü (2) die Möglichkeiten von Export, Auf- und Zuklappen der Rollenprofile, mehrspaltige Sortierung und Spaltenauswahl und Reihenfolge der Spalten der angezeigten Tabelle.
