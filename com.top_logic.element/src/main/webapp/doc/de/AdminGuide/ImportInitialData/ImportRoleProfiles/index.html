Die Berechtigungen für Rollen der gesamten Berechtigungsstruktur, die im Rollenprofil dargestellt sind, können dort gesichert und wieder eingespielt werden. Im Reiter "Administration : Berechtigungen   : Rollenprofile" kann die Berechtigungssturktur durch anklicken des Burger-Menüs (1)   exportiert werden. Zum einen nach "XML", zum anderen nach "Excel". Zu Beachten ist, dass nur ein "XML"-Export (2) auch wieder eingespielt werden kann; mit Excel ist dies nicht möglich.

 

 

Abbildung 1:   Einspielen der Rollenprofile

  • Wechseln in den Reiter Administration : Berechtigungen : Rollenprofile.
  • Dort das Kommando "Importieren..." auswählen (vgl. Abbildung 1)
  • Die Datei "security.xml" auswählen.
  • "Importieren" drücken.

Jetzt werden die Berechtigungsprofile für die Anwendung installiert. Der Import war erfolgreich, wenn der Dialog geschlossen wird und keine Fehlermeldungen angezeigt werden.
