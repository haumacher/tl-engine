Syntax

	resolveAlias($alias)

Beschreibung

Löst alle Auftreten eines Aliases mit durch die Anwendungskonfiguration angegebenen Werten auf, wenn diese nicht schon von der Anwendungskonfiguration übernommen wurden.

Parameter

Name Typ Beschreibung Pflicht Default
alias Zeichenkette Der Names des Alias, der aufgelöst werden soll. ja

Rückgabewert

Typ: Zeichenkette

Den aufgelösten Alias.

Beispiele

	 resolveAlias('%HOST%:%PORT%')

Ausgabe: localhost:8090

Unter der Voraussetzung, dass für den Alias %HOST% in der Konfiguration localhost und für %PORT% 8090 angegeben wurde.
