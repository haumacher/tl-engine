Syntax

	$expr1.compare($expr2)

Beschreibung

Liefert das Vergleichsergebnis von expr1 und expr2. Das Vergleichsergebnis ist kleiner als 0, gleich 0, größer als 0, wenn expr1 kleiner als, gleich, oder größer als expr2 ist.

Parameter

Name Typ Beschreibung Pflicht Default
expr1 Zahl/Zeichenkette/Wahrheitswert/Menge Ein Element, das mit expr2 verglichen werden soll. Siehe Vergleichsoperatoren. ja
expr2 Zahl/Zeichenkette/Wahrheitswert/Menge Ein Element, das mit expr1 verglichen werden soll. Siehe Vergleichsoperatoren. ja

Rückgabewert

Typ: Zahl

Abhängig von Ergebnis liefert die Funktion:

  • 0, wenn expr1 und expr2 gleich groß sind
  • eine Zahl größer 0, wenn expr1 größer ist als expr2
  • eine Zahl kleiner 0, wenn expr1 kleiner ist als expr2

Beispiele

Zahlen

	compare(1, 5)

Ausgabe: -1

Zeichenketten

	compare("z", "a")

Ausgabe: 25

Beim Vergleich von Zeichenketten wird nicht nur eine Zahl kleiner, größer oder gleich 0 ausgegeben, sondern noch zusätzlich, wie weit die Zeichenketten voneinander entfernt sind.

Wahrheitswerte

	compare("true", "false")

Ausgabe: 1

Listen

	compare(list(3 , 2, 1) , list(1, 2, 3))

Ausgabe: 1

Listen werden elementweise miteinander verglichen, bis bei einem Elementpaar ein Element größer oder kleiner ist, vgl. Vergleichsoperatoren.

Sortieren mit einem Vergleichswertergebnis

list("z", "a", "f")
  .sort(x -> y -> compare($x, $y))

Ausgabe: Eine Liste mit den Elementen ["a", "f", "z"].

Vgl. mit dem Beispiel "Vergleich mit Vergleichsfunktion" in Sortieren.
