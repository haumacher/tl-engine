Syntax

response(
  statusCode: $statusCode, 
  content: $content, 
  contentType: $contentType
)

Beschreibung

Erstellt ein Antwort-Objekt, dass auf eine HTTP-Anfrage zurückgegeben wird.

Parameter

Name Typ Beschreibung Pflicht Default
statusCode Zahl Status-Code für die Antwort. Hier werden HTTP-Response Codes angegeben, z.B. 200 (OK) oder 404 (NOT FOUND). ja
content
Objekt
Inhalt der Antwort. ja
contentType Zeichenkette
Inhaltstyp der Antwort. Dieser muss zum Resultat der Antwort passen.
nein application/json

Beispiele

Beispiel 1

	response(404, {"message":"Element could not be found"});

Beispiel 2

	response(
  500,
  "<html>
    <head><title>Error 500 Server Error</title></head>
    <body><h2>HTTP ERROR 500 Server Error</h2></body>
   </html>",
  "text/html");
