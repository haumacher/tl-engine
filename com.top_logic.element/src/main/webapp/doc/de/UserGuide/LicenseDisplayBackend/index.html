Frontend-Komponenten

Frontendkomponenten sind solche Komponenten, die durch Aufruf der Anwendung zum Client (also zum Webbrowser des Benutzers) übertragen und dort im Hauptspeicher oder im Dateisystem (temporär) gespeichert werden.

TopLogic ermöglicht jedem Benutzer die Anzeige und den Download der Lizenzinformationen zu diesen Komponenten:

Abbildung 1: Info Symbol oben links

Abbildung 2: Info Symbol unten rechts

Je nach verwendetem Theme finden Sie ein „Info“ – Symbol entweder im Anwendungsfenster oben links oder in der Fußleiste unten rechts.

Durch Klick auf dieses Symbol werden die Lizenzinformationen zu verwendeten Drittanbieter Bibliotheken angezeigt. Der Lizenztext jeder Lizenz kann mit einem Klick auf den Lizenznamen heruntergeladen werden.

Backend-Komponenten

Backend Komponenten sind Teil der installierten Applikation, werden jedoch nicht zum Client übertragen.

Abbildung 3: Anzeige der Informationen zu Drittanbieterbibliotheken.

Informationen zu Drittanbieterbibliotheken können im Reiter "Administration : Monitor : Monitor" im Burger-Menu -->   „Drittanbieterbibliotheken anzeigen“ abgerufen werden.

Der Lizenztext jeder Lizenz kann mit einem Klick auf den Lizenznamen heruntergeladen werden.
