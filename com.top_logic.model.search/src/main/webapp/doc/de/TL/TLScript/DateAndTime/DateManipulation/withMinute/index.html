Syntax

	$calendar.withMinute($minute)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Minute auf minute gesetzt wird.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Minute geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
minute Zahl Wert für die neue Minute, die für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Minute.

Beispiele

	dateTime(2012, 8, 15, 5, 55, 24, 45).toUserCalendar().withMinute(22).minute()

Ausgabe: 22
