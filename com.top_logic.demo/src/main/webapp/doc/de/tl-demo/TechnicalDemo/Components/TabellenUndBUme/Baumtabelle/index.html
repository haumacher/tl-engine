Die Sicht stellt die Demotypen in einer Baumtabelle dar. Die Konfiguration entspricht in großen Teilen die von Baum.

Zusätzlich kann man die unterstützten Typen der Tabelle, die Spalten, die standardmäßig in der Tabelle dargestellt werden sollen und die ID-Spalte, also die Spalte, die die Baumstruktur definiert, konfiguriert werden.
