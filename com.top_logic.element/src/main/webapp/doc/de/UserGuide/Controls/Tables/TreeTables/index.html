Mit Baumtabellen werden Inhalte dargestellt die eine Hierarchie besitzen. Das Besondere an Baumtabellen ist, dass man Elemente, die eine Unterstruktur besitzen, auf- und zuklappen kann.

Das Auf-   bzw. Zuklappen   kann für jedes Element einzeln ausgeführt werden.

Über das Burger-Menü kann der Benutzer alle Baumstrukturen aufklappen   oder alle zuklappen .

Neben der Baumstruktur ist meistens eine Detailsicht eingeblendet, in der der Nutzer auch Änderungen vornehmen kann.

Wie auch schon im Absatz "Anpassen von Tabellen" beschrieben, lassen sich auch hier die Fenster minimieren und maximieren.
