Syntax

	response($status_code, $result, $content_type)

Beschreibung

Erstellt ein Antwort-Objekt dass auf eine HTTP-Anfrage zurückgegeben wird

Parameter

Name Typ Beschreibung Pflicht Default
status_code Zahl Status-Code für die Antwort. Hier werden HTTP-Response Codes angegeben, z.B. 200 (OK) oder 404 (NOT FOUND) ja
result
Objekt
Inhalt der Anwort ja
content_type Zeichenkette
Inhaltstyp der Antwort. Muss zur Antwort passen
nein application/json

Beispiele

Beispiel 1

	response(404, {"message":"Element could not be found"});

Beispiel 2

	response(
  500,
  "<html>
    <head><title>Error 500 Server Error</title></head>
    <body><h2>HTTP ERROR 500 Server Error</h2></body>
   </html>",
  "text/html");
