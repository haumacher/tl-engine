Dazu muss man nur den Knoten von ConfigPart ableiten, dann bedeutet die Annotation @Container an einem der Getter, dass er den Vaterknoten liefern soll:

public interface Latter extends ConfigurationItem {

	List<Step> getSteps();

	public interface Step extends ConfigPart {

		public static final String LATTER = "latter";

		/** The container. */
		@Name(LATTER)
		@Hidden
		@Container
		Latter getLatter();
	}
}
