Normalerweise haben Ausdrücke in TL-Script keine Seiteneffekte. D.h. dass die Auswertung eines TL-Script Ausdrucks die Daten der Anwendung nicht verändert. Um Transaktionen beschreiben zu können, gibt es aber einige Operationen, deren Auswertung Anwendungsdaten modifiziert. Dazu gehört das Erzeugen, Löschen und Verändern von Objekten der Anwendung. Solche zustandsverändende Operationen können nur in speziell ausgezeichneten Stellen bei der Anwendungskonfiguration verwendet werden. I.d.R. heißen solche Konfigurationsoptionen "Operation".
