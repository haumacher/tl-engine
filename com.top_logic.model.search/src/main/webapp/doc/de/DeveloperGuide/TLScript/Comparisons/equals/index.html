Syntax

	$e1 == $e2

$e1.equals($e2)

Beschreibung

Boolscher Ausdruck, der als wahr bewertet wird, wenn e1 und e2 zum gleichen Wert ausgewertet werden.

Parameter

Name Typ Beschreibung Pflicht Default
e1 Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Objekt, das mit e2 verglichen werden soll. ja
e2 Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Objekt, das mit e1 verglichen werden soll. ja

Rückgabewert

Typ: Wahrheitswert

true, wenn das Ergebnis der Auswertung besagt, dass die beiden Objekte identisch sind.

Beispiele

Zahlen

	1 == 1

Ausgabe: true

Gleiche Mengen

	list(1, 2).equals(list(1, 2))

Ausgabe: true

Ungleiche Mengen

	list(1, 2).equals(list("one", "two"))

Ausgabe: false
