Syntax

$resKey.localize(lang: $language)

Evaluates an internationalized string in a given language.

Parameters

Name Type Description Mandatory Default
resKey Internationalized string The internationalized text whose localization is to be determined. yes
language String The code of the language to which the localization is to be made. no The language of the user.

Return value

Type: string

The value that a user with the selected language will see.

Examples

{
  resKey = {"de": "Hallo", "en": "Hello"}.internationalize();

  $resKey.localize("en"); // Hello
  $resKey.localize("de"); // Hallo
}
