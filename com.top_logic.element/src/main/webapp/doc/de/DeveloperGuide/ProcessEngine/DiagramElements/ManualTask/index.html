Eine manuelle Aufgabe repräsentiert eine Arbeitssicht im System. Ihre Eigenschaften können über die Selektion ihres Diagramm-Symbols erreicht werden.

Erreicht ein Prozess eine manuelle Aufgabe, erhalten sie diejenigen Nutzer, welchen diese Aufgabe zugewiesen ist (vgl. Bahn), in ihre Aktuelle-Aufgaben-Kachel in ihrem Prozess-Cockpit. Wählt ein Nutzer die Aufgabe zur Bearbeitung aus, wird ihm eine Eingabe-Maske präsentiert, welche über einen Formular-Editor in den Eigenschaften der Aufgabe definiert werden kann.

Die Arbeitssicht für eine Aufgabe erhält automatisch eine Funktionalität, um die Aufgabe abzuschließen. Beim Abschluss wählt entweder der Bearbeiter der Aufgabe den nächsten Prozessschritt aus den möglichen Folgeaufgaben aus, oder das System wählt automatisch, wenn für eine Prozessweiche Regeln hinterlegt sind (vgl. Entscheidung).
