Syntax

	$date.toSystemCalendar()

Beschreibung

Transformiert einen Datumswert zu einen Systemkalender um auf Felder wie Jahr, Monat, etc. zugreifen und diese manipulieren zu können.

Es wird erwartet dass der Datumswert vom Typ tl.core:Date ist.

Parameter

Name Typ Beschreibung Pflicht Default
date Fachobjekt Ein Datumswert, der durch den Kalender repräsentiert werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Systemkalender, der den Datumswert von date repräsentiert.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

	toSystemCalendar(date(2012, 5, 5))

Ausgabe: (System-) Kalenderobjekt
