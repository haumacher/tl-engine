Syntax

	$calendar.day()

Beschreibung

Der Tag des gegebenen Kalenders.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Tag ermittelt werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja

Rückgabewert

Typ: Zahl

Eine Zahl, die den Tag des gegebenen Kalenders repräsentiert.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

	date(2012, 8, 5).toSystemCalendar().day()

Ausgabe: 5
