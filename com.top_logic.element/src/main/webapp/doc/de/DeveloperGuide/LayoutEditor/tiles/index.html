Im Designmodus kann ein Kachelcockpit erstellt werden.

Hier muß als Einstiegspunkt Kacheln konfiguriert .

Unterhalb dieser Wurzel-Kachel gibt es  verschiedene Möglichkeiten sich eine Kachel-Hierarchie zusammenzustellen. Es kann zwischen verschiedenen kachelspezifischen Komponenten gewählt werden, die wieder beliebig geschachtelt werden können:

  • Kachelgruppe
  • Kontextkomponenten in Form von
    • Kontext-Tabelle
    • Kontext-Baum
    • Kontext-Liste
