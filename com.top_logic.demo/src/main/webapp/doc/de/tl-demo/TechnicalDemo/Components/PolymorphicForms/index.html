In der Sicht können Objekte der Typen (A, B, C) über einen Anlagedialog erzeugt, in der Tabelle angezeigt und die Eigenschaften in einem nebenstehenden Formular bearbeitet werden. In der Sicht kann getestet werden, wie sich die Objektanlage mit Typauswahl und die Formularanzeige mit unterschiedlichen Objekttypen verhält.
