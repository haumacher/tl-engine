Syntax

	resolveModelPart($partName)

Beschreibung

Löst ein Modellelement über den voll qualifizierten AttributenamenpartName auf.

Parameter

Name Typ Beschreibung Pflicht Default
partName Zeichenkette Der voll qualifizierte Name des Modellelements das aufgelöst werden soll ja

Rückgabewert

Typ: Fachobjekt

Das Modellelement mit dem voll qualifizierten NamenpartName.

Beispiele

Einfache Attribut-Auflösung

{
moduleName = "my.module"; className = "MyType"; localAttrName="myAttribute";
resolveModelPart($moduleName + ":" + $className + "#" + $localAttrName);
}

Ausgabe: `my.module:MyType#myAttribute`

Einfache Typ-Auflösung

{
moduleName = "my.module"; className = "MyType";
resolveModelPart($moduleName + ":" + $className);
}

Ausgabe: `my.module:MyType`
