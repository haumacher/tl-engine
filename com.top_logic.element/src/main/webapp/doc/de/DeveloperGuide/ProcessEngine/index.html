Mit der in TopLogic eingebauten Workflow-Engine ist es möglich, die Interaktion zwischen unterschiedlichen Nutzern bzw. Nutzergruppen in einer Anwendung über ein BPMN-Diagramm zu beschreiben. Jedes hochgeladene Diagramm, bzw. jeder angelegte Workflow repräsentiert dabei eine Mini-App, die über die Startseite der Anwendung aufgerufen werden kann.

Eine solche Prozess-App besteht aus einem personalisierten Cockpit, welches je nach Rolle des angemeldeten Nutzers das Starten von Prozessen und die Bearbeitung von in diesem Prozess anfallenden Aufgaben ermöglicht.

Das Prozessmodell steuert dabei die Zuweisung von Aufgaben an die im Prozess definierten Rollen. Jede von einem Nutzer zu übernehmende Aufgabe repräsentiert sich in einer Eingabemaske, welche dem Nutzer präsentiert wird, der eine Aufgabe zur Bearbeitung ausgewählt hat. Darüberhinaus kann die Prozess-Engine automatische Aufgaben übernehmen, für die keine Nutzerinteraktion notwendig ist. Zu solchen automatischen Aufgaben gehört das Versenden von Nachrichten und das Ausführung von Skripten.
