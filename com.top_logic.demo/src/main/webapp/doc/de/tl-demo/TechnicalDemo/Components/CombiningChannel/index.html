Die Kanalkombination ist eine spezielle Form der Kanaltransformation mit mehreren Eingangskanälen. Die hier dargestellte sicht zeigt die Tabelle alle Demo-A-Objekte an, welche zwei bestimmte Klassifikationselemente in ihren Attributen "checklistMulti" und "checklistSingle" besitzten. Diese Filterung wird hier nicht über einen nutzerdefinierten Tabellenfilter realisiert, sondern über zwei der Tabelle vorgeschaltete explizite Auswahlkomponenten.

Die hier angezeigetn Komponenten sind so verschaltet, dass die Zieltabelle ihr Kontext-Modell als "Wertekombination" der Selektionskanäle der beiden Selektionskomponenten bestimmt.

Diese Wertekombination bewirkt, dass das Kontextobjekt der Tabelle eine zweielementige Liste ist, welche an Position 0 das im linken Selektor selektierte Objekt enthält und an Position 1 das im rechten Selektor selektierte Objekt.

Diese Wertekombination wird in der Funktion, welche die Zeilenobjekte berechnet, für eine doppelte Filterung der Demo-A-Objekte nach den beiden Kriterien verwendet.
