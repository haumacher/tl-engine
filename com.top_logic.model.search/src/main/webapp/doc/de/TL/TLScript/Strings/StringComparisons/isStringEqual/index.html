Syntax

	$string1.isStringEqual($string2)

$string1.isStringEqual($string2, $case-sensitive)

Beschreibung

Vergleicht die Zeichenketten string1 und string2 auf Gleichheit.

Parameter

Name Typ Beschreibung Pflicht Default
string1 Zeichenkette Eine Zeichenkette, bei der geprüft wird, ob sie mit string2 übereinstimmt ja
string2 Zeichenkette Eine Zeichenkette, bei der geprüft wird, ob sie mit string1 übereinstimmt ja
case-sensitive Wahrheitswert Bestimmt, ob auch die Groß- und Kleinschreibung berücksichtigt werden soll. nein false - die Groß- und Kleinschreibung wird nicht berücksichtigt.

Rückgabewert

Typ: Wahrheitswert

Ob string1 string2 entspricht.

Beispiele

Einfacher Vergleich

	isStringEqual("Hello world!", "Hello world!")

Ausgabe: true

Casesensitiver Vergleich 1

	isStringEqual("Hello world!", "Hello world!", true)

Ausgabe: true

Casesensitiver Vergleich 2

	isStringEqual("Hello world!", "hello world!", true)

Ausgabe: false
