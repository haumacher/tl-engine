Zentraler Bestandteil in TopLogic ist die Anzeige der Daten in einer Tabelle. Um das angezeigte Tabellenergebnis besser bearbeitbar zu machen kann der angezeigte Inhalt der Tabelle durch hinzufügen und abwählen von Spalten beliebig angepasst werden.

Um die Tabellenspalten anzupassen, wird über das Burger-Menü (1) der Unterpunkt "Spaltenauswahl und -reihenfolge" ausgewählt. Im angezeigten Popup-Window kann dann der Tabelle jedes angezeigte Attribut hinzugefügt oder abgewählt werden (2).
