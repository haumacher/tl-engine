Syntax

	$calendar.withAmPm($value)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Tageszeit (a.m. oder p.m.) auf amPm gesetzt wird. Durch das Verändern der Tageszeit können sich auch andere Werte wie zum Beispiel die Stunde verändern.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Tageszeit geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
amPm Zahl Wert für den Vormittag a.m. (0) und den Nachmittag p.m. (1). ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Tageszeit.

Beispiele

Ändern von amPm

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withAmPm(1).amPm()

Ausgabe: 1

Anpassung der Stunde nach amPm Anpassung

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withAmPm(1).hour()

Ausgabe: 19

Durch das Umstellen auf p.m. steht 7 nun für 19 Stunden.
