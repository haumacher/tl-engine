Die Reihenfolge der Tabs in einer Tableiste kann nachträglich angepasst werden. Hierfür muss im Design-Modus die Option "Tabs konfigurieren" aus dem Burger-Menü der jeweiligen Tableiste oder der Sidebar gewählt werden.

In dem sich daraufhin öffnenden Auswahldialog kann die Reihenfolge der dargestellten Tabs angepasst oder weitere verfügbare Tabs ausgewählt und in die Tableiste der aktuellen Sicht eingefügt werden.
