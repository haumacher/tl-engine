Syntax

	$expr.toList()

Beschreibung

Wandelt das Ergebnis der Auswertung von expr in eine Liste um. Folgende Fälle können dabei auftreten:

  1. expr ist bereits eine Liste: expr wird zurückgegeben
  2. expr ist eine beliebige Kollektion (z.B. Menge): expr wird in eine Liste umgewandelt.
  3. expr ist null: Es wird eine Leere Menge zurückgegeben.
  4. Für alle anderen Werte wird eine ein-elementige Menge erstellt, vlg. Einpacken

Parameter

Name Typ Beschreibung Pflicht Default
expr Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Ein Ausdruck, dessen Auswertung in eine Liste umgewandelt werden soll. ja

Rückgabewert

Typ: Aufzählung

Eine Liste, die die Werte aus expr repräsentiert.

Beispiele

Umwandlung einer Liste

	list(1, 1, 2, 3, 3).toList()

Ausgabe: Eine Liste mit den Elementen [1, 1, 2, 3, 3]

Da es sich schon um eine Liste handelt, wird diese einfach zurückgegeben.

Umwandlung eines Einzelwerts

	toList(1)

Ausgabe: Eine Liste mit dem Element [1].

Umwandlung von null

	toList(null)

Ausgabe: Eine leere Liste
