Testfall für die Überschreibung von Referenzen mit Berechnungsvorschriften. Im Modul test.override gibt es einen abstrakten Obertyp I, der eine Referenz root definiert. Diese Referenz wird in den Untertypen A und R jeweils mit einer Berechnungsvorschrift überschrieben. Nur der Untertyp S erbt die Referenz unverändert und erlaubt eine Werteingabe an der Benutzeroberfläche. Der Typ R ist gleichzeitig der Zieltyp der getesteten Referenz root.

In der "Tabelle mit Instanzen" können über einen Dialog mit generischem Formular und einen Dialog mit für den Typ spezialisierten Formular jeweils Instanzen der drei Typen angelegt werden. Nach der Anlage können die Instanzen in den beiden nebenstehenden Formularen jeweils mit generischem Forumuar und spezialisiertem Formular betrachtet werden.

Beim Test ist wichtig, dass eine berechnete Überschreibung der root-Referenz nicht bearbeitet werden kann. In den Formularen neben der Tabelle und in der Tabelle selbst muss der korrekte Wert der Berechnung angezeigt werden.
