In einer Kachelgruppe werden, ebenso wie bei Tabs, mehrere Komponten konfiguriert und angezeigt. Im Gegensatz zu den Reitern werden diese Komponenten als Kacheln angezeigt.

Es gibt mehrere Arten von Kacheln die zu einer Kachelgruppe hinzugefügt werden können:

 

Standard Kachel

In einer Standard Kachel können beliebige Komponenten konfiguriert werden.

Vorgefertigte Kacheln

Es gibt einige Kacheln, bei denen die Vorschau der Kachel und die angezeigte Komponente vorgegeben ist.

Chart JS Kachel

Hier kann eine Auswertung über ein Modell vorgenommen werden. Diese Auswertung wird in der Kachel und in der Vorschau für diese Kachel angezeigt.

Dokumentenkachel

In der Dokumentenkachel wird ein PDF-Dokument angezeigt. Welchese Dokument angezeigt werden soll, wird über einen TL-Script Ausdruck angegeben. Das Dokument wird ebenfalls in der Vorschau der Kachel angezeigt.

Objekt-Kacheln

Hier wird eine alternative Darstellung für eine Menge von Fachobjekten konfiguriert. Die Fachobjekte die angezeigt werden, werden mit einem TL-Script Ausdruck angegeben. Für jedes dieser Objekte wird eine eigene Kachel angezeigt. Bei Klick auf eine Kachel kann dieses Objekt als Kontextmodell für eine weitere Komponente benutzt werden.
