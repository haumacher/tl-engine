Für eine neu erstellte Anwedung ist es wichtig, dem Nutzer eine Dokumentation anzubieten, welche Sichten für welchen Zweck benutzt werden können. Die Dokumentationsseiten für die einzelnen Sichten müssen nicht manuell angelegt werden. Stattdessen kann in Administration > Development > Hilfe-Editor das Kommando Hilfestruktur generieren benutzt werden.

Diese Kommando generiert unterhalb der ausgewählten Seite Hilfeseiten basierend auf der Anwendungsstruktur. Für jede Sicht in der eine Hilfe-ID gesetzt ist, wird eine Seite erstellt. Nötigenfalls werden auch übergeordnete Seiten erstellt um die Tab- und Dialog-Struktur der Anwendung nachzubilden.

Das Kommando "Hilfestruktur generieren" kann mehrfach ausgeführt werden; es werden nur Seiten für Sichten erstellt die noch nicht existieren. Bereits erstellte Seiten werden nicht geändert.
