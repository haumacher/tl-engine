Syntax

	resolveModelPart($partName)

Description

Resolves a model element using the fully qualified attribute namepartName.

Parameters

Name Type Description Mandatory Default
partName String The fully qualified name of the model element to be resolved. yes

Return value

Type: Business object

The model element with the fully qualified namepartName.

Examples

Simple attribute resolution

{
moduleName = "my.module"; className = "MyType"; localAttrName="myAttribute";
resolveModelPart($moduleName + ":" + $className + "#" + $localAttrName);
}

Output: `my.module:MyType#myAttribute`

Simple type resolution

{
moduleName = "my.module"; className = "MyType";
resolveModelPart($moduleName + ":" + $className);
}

Output: `my.module:MyType`
