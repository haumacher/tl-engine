Diese Sicht demonstriert Dokumentenattribute von Objekten, die in einer Kompositionstabelle im Kontext eines anderen Objektes dargestellt und bearbeitet werden können.

Hierfür definiert das Modul `test.webfolder` einen Kontext-Typ, der hier als Kachel dargestellt wird. Dieser enthält mehrere Teilobjekte, die selbst wiederum neben einem Namen ein Dokumentenattribut besitzen.
