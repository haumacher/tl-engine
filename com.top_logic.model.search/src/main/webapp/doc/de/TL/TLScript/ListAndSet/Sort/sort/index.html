Syntax

	$set.sort()

$set.sort($comparator)

Beschreibung

Sortiert die Menge set. Die Reihenfolge der Sortierung kann durch die Vergleichsumkehr umgekehrt werden.

Parameter

Name Typ Beschreibung Pflicht Default
set Menge Eine Menge von Elementen, die sortiert werden sollen. ja
comparator Funktion Eine Vergleichsfunktion, mit deren Hilfe die Elemente sortiert werden sollen. Die Vergleichsfunktion kann aus einer Zugriffsfunktion erzeugt werden, siehe Vergleichsfunktion. nein Die Elemente werden direkt miteinander verglichen. Das ist allerdings nur sinnvoll für primitive Werte, für die ein kanonischer Vergleich definiert ist. Objekte sind nicht direkt miteinander vergleichbar und brauchen daher eine Vergleichsfunktion. Eine Vergleichsfunktion erwartet zwei Argumente x und y. Als Ergebnis erhält man einen Wert kleiner, gleich oder größer als 0, falls x kleiner, gleich oder größer als y ist.

Rückgabewert

Typ: Menge

Die neu Sortierte Menge set.

Beispiele

Einfacher Vergleich mit Zahlen

	list(8, 10, 5, 6).sort()

Ausgabe: Eine Menge mit den Elementen [5, 6, 8, 10].

Vergleich mit Vergleichsfunktion

	list("z", "a", "f")
  .sort(x -> y -> {
    if ($x < $y, 
      -1, 
      if ($x > $y, 1, 0));
  })

Ausgabe: Eine Liste mit den Elementen ["a", "f", "z"].

Dieses Beispiel könnte auch mit einem Vergleichswert vereinfacht werden.
