In den Bearbeitungsdialogen werden in der Regel die Buttons "Übernehmen" und "Speichern" angezeigt. Durch Drücken eines der beiden Buttons werden die zuvor auf Korrektheit geprüften Inhalte (Änderungen) in die Datenbank geschrieben. Im Gegensatz zu "Speichern" bleibt bei "Übernehmen" der Bearbeitungsdialog im Editiermodus. Beim "Speichern" wird in den Lesemodus zurückgestellt.

Dies hat Auswirkung für die Handhabung der Bearbeitungsdialoge, so können Eingabe ohne Probleme zwischengepeichert werden.
