Syntax

	$calendar.withDayOfYear($day)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei Tag des Jahres auf day gesetzt wird. Durch das Verändern des Tages können sich auch andere Werte wie zum Beispiel der Monat ändern.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Tag des Jahresgeändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
day Zahl Wert für den neuen Tag des Jahres, der für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für den Tag des Jahres.

Beispiele

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withDayOfYear(200).dayOfYear()

Ausgabe: 200
