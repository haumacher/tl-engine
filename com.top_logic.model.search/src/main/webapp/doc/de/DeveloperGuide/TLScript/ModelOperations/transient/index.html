Syntax

	$object.transient()

Beschreibung

Prüft, ob ein Objekt als transient erzeugt wurde. Transiente Objekte sind nur lokal in der Session des Benutzers sichtbar und können nicht in einer Referenz eines persistenten Objektes gespeichert werden.

Parameter

Name Typ Beschreibung Pflicht Default
object Fachobjekt Das zu testende Objekt. ja

Rückgabewert

Typ: Wahrheitswert

true, wenn das Objekt transient ist.

Beispiele

Transiente Objekte

{
    object = new(`my.module:MyClass`, transient: true);
    $object.transient();
}

Ausgabe: true

Persistente Objekte

{
    object = new(`my.module:MyClass`);
    $object.transient();
}

Ausgabe: false
