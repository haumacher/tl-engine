Eine Grid ist eine spezielle Tabelle, in der Daten direkt zeilenweise bearbeitet werden können. Aktiviert man den Bearbeitenmodus , können Inhalte bearbeitet werden. Wird nach dem Eingeben von Daten eine andere Zeile selektiert, wird die zuletzt bearbeitete Zeile gespeichert und die neue Zeile in den Bearbeitungsmodus geschaltet.

 

Abbildung 1: Grid-Tabelle im Bearbeiten-Modus

Im Grid gibt es zusätzlich die Möglichkeit, einzelne Elemente über den Button Öffnen   in einem Dialog zu bearbeiten. Dies erlaubt die Veränderung von zusätzlichen Daten, welche in der Tabelle nicht direkt bearbeitet werden können.
