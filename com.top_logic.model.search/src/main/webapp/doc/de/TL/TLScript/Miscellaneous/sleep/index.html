Syntax

	sleep($timeInMillis)

Beschreibung

Hält die Ausführung des Scriptes für die konfigurierte Zeit an.

Parameter

Name Typ Beschreibung Pflicht Default
timeInMillis Zahl Zeit in Millisekunden die gewartet werden soll ja

Beispiele

Beispiel 1

	sleep(5000);

Die Ausführung des TL-Scriptes wird für 5 Sekunden gestoppt.
