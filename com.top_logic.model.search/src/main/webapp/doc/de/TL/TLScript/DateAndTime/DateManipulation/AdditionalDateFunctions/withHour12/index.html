Syntax

	$calendar.withHour12($hour)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Stunde im 12-Stunden Format auf hour gesetzt wird. Durch das Verändern der Stunde können sich auch andere Werte wie zum Beispiel die Stunde (24-Stunden Format) verändern.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Stunde geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
hour Zahl Wert für die neue Stunde, der für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Stunde.

Beispiele

hour <=12

	dateTime(2012, 8, 15, 7, 55, 24, 13).toUserCalendar().withHour12(2).hour12()

Ausgabe: 2

hour > 12

	dateTime(2012, 8, 15, 7, 55, 24, 13).toUserCalendar().withHour12(15).hour12()

Ausgabe: 3

Die Stunde wird ins 12-Stunden-Format umgewandelt.

hour() nach withHour12()

	dateTime(2012, 8, 15, 7, 55, 24, 13).toUserCalendar().withHour12(15).hour()

Ausgabe: 15

Setzt man einen neuen Wert für hour12, so wird auch hour geändert.
