Syntax

	$set1.containsSome($set2)

Beschreibung

Prüft, ob die Menge set1 mindestens ein Element der Menge set2 enthält.

Parameter

Name Typ Beschreibung Pflicht Default
set1 Menge Eine Menge, für die überprüft werden soll, ob mindestens ein Element aus set2 enthalten ist. ja
set2 Menge Eine Menge, für die überprüft werden soll, ob mindestens ein Element in set1 enthalten ist. ja

Rückgabewert

Typ: Wahrheitswert

true, wenn mindestens ein Element aus set2 in set1 enthalten ist.

Beispiele

Menge enthält Element

	list(1, 2, 3, 4).containsSome(list(1, 2))

Ausgabe: true

Menge enthält keins der Elemente

	list(1, 2, 3, 4).containsSome(list(5, 6, 7))

Ausgabe: false
