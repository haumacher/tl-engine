Syntax

	$calendar.withDstOffset($offset)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei der Sommerzeit-Offset auf offset gesetzt wird. Durch das Verändern des Offsets können sich auch andere Werte wie zum Beispiel die Stunde ändern.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Sommerzeit-Offset geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
offset Zahl Wert für den neuen Sommerzeit-Offset in Millisekunden, der für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für den Sommerzeit-Offset.

Beispiele

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withDstOffset(3600000).dstOffset()

Ausgabe: 3.600.000
