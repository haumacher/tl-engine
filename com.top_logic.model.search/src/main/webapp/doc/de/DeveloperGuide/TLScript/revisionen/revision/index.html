Syntax

	$object.revision()

Beschreibung

Ermittelt die Revision in der ein Fachobjekt vorliegt.

Parameter

Name Typ Beschreibung Pflicht Default
object Fachobjekt Das Fachobjekt für das die Revision ermittelt werden soll ja

Rückgabewert

Typ: Revision

Die Revision des Fachobjektes, wenn es ein historischer Stand ist oder CURRENT für ein aktuelles Objekt.

Beispiele

Beispiel 1

	$historicObject.revision()

Ausgabe: r42

Wenn das gegebene Fachobjekt historicObject in Revision r42 vorliegt.

Beispiel 2

	$currentObject.revision()

Ausgabe: CURRENT

Wenn das gegebene Fachobjekt currentObject kein historisches, sondern ein aktuelles Objekt ist.
