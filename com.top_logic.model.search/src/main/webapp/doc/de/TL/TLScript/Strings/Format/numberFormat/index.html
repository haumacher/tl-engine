Syntax

	numberFormat($pattern)

Beschreibung

Konstruiert ein Zahlenformat mit dem angegebenen Muster für entweder parse() oder format() Aufrufe.

Siehe DecimalFormat.

Parameter

Name Typ Beschreibung Pflicht Default
pattern Zeichenkette

Das Pattern in dem formatiert werden soll. Es wird wie folgt angegeben:

  • 0 – eine Zahl, falls vorhanden, sonst 0
  • # – eine Zahl, falls vorhanden, sonst nichts
  • . - Gibt an, wo das Dezimalstrennzeichen platziert werden soll
  • , - Gibt an, wo das Gruppierungstrennzeichen platziert werden soll
  • für weitere Symbole siehe DecimalFormat
 ja

Rückgabewert

Typ: Fachobjekt

Ein Zahlenformat, das nun für parse() oder format() Aufrufe verwendet werden kann.

Beispiele

Format mit #

	numberFormat("###,###.###").format(123456.59874)

Ausgabe: 123.456,599

Format mit # und 0

	numberFormat("###,###.000").format(123456.5)

Ausgabe: 123.456,500

Weniger Zeichen

	numberFormat("###,###.###").format(123456.5)

Ausgabe: 123.456,5
