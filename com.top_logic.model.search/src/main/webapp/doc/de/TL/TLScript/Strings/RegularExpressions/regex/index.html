Syntax

	regex($string)

Beschreibung

Erzeugt ein Regular-Expression-Pattern aus dem gegebenen String. Das Pattern kann über die Ausdrücke regexSearch() und regexReplace() auf Zeichenketten angewendet werden.

Parameter

Name Typ Beschreibung Pflicht Default
string Zeichenkette Ein Regex Ausdruck, aus dem das Pattern erstellt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Regular-Expression-Pattern, das auf Zeichenkette angewendet werden kann.

Beispiele

Buchstaben und Zahlen

	regex("[A-Za-z0-9]")

Ausgabe: Pattern für einen beliebigen lateinischen Buchstaben oder eine beliebige Ziffer.

Zahlen

	regex("[0-9]")

Ausgabe: Pattern für eine beliebige Ziffer.
