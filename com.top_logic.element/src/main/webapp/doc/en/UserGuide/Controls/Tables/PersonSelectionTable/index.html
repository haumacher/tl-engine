==Anpassbare Tabelle==
Anpassbare Tabellen zeichnen sich durch verschiedene Funktionen aus:
* '''Sortierung''' (aufwärts/abwärts) der Tabelleninhalte bezogen auf eine ausgewählte Spalte kann durch Klicken auf den Spaltenkopf oder Aufrufen der Funktion [[de/Funktionen#Konfigurationsdialog für Spaltensortierung öffnen|Konfigurationsdialog für Spaltensortierung öffnen]] vorgenommen werden
* Ändern der '''Spaltenreihenfolge''' kann durch Ziehen einer Spalte an die gewünschte Stelle oder Verwenden der Funktion [[de/Funktionen#Konfigurationsdialog für Spaltenauswahl und Reihenfolge öffnen|Konfigurationsdialog für Spaltenauswahl und Reihenfolge öffnen]] erfolgen.
* Es lassen sich beliebig viele '''Spalten fixieren''', so dass diese bei einer Tabelle, die breiter ist als der sichtbare Bereich, beim Scrollen an ihrer Position bleiben. Dazu ist der breite Auswahlbalken an die gewünschte Stelle zu ziehen.
* Es stehen [[de/Funktionen#Spaltenfilter|'''Filter''']] zur Verfügung, mit denen sich Inhalte einer Tabelle bezogen auf die ausgewählte(n) Spalte(n) filtern lassen.
