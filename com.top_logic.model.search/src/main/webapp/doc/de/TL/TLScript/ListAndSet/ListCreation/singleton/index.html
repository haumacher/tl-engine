Syntax

	$expr.singleton()

Beschreibung

Packt das Ergebnis von expr in eine einelementige Menge ein. Ein nicht null-Wert resultiert in einer Menge der Größe eins mit diesem Wert als einzigem Element, auch wenn es sich bei expr selbst um eine Menge handelt. Ein null Wert resultiert in der leeren Menge.

Siehe auch Auspacken.

Parameter

Name Typ Beschreibung Pflicht Default
expr Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Ein Ausdruck, dessen Auswertung in eine ein-elementige Menge verpackt werden soll. ja

Rückgabewert

Typ: Menge

Eine Menge mit einem Element, das das Ergebnis von expr repräsentiert. Eine leere Menge, wenn null für expr benutzt wird.

Beispiele

Einpacken eines Elements

	singleton(5)

Ausgabe: Eine Menge mit dem Inhalt [5].

Einpacken von null

	singleton(null)

Ausgabe: Eine leere Menge.

Einpacken einer Liste

	singleton(list(1, 2, 2))

Ausgabe: Eine Menge mit dem Inhalt [1; 2; 2], wobei es nur ein Element ist.
