Syntax

	$object.inCurrent()

Beschreibung

Ermittelt für ein gegebenes Fachobjekt die akutelle Version. Wenn das Fachobjekt nicht mehr existiert, wird null herausgegeben.

Parameter

Name Typ Beschreibung Pflicht Default
object Fachobjekt Fachobjekt für das die aktuelle Version ermittelt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Die aktuelle Version des Fachobjektes, oder null, wenn das Fachobjekt aktuell nicht existiert.
