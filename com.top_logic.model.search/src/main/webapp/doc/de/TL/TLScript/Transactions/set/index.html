Syntax

	$object.set(`$attribute`, value)

Beschreibung

Aktualisiert den Wert des Attributs attribute des Objektes object zu value.

Parameter

Name Typ Beschreibung Pflicht Default
object Fachobjekt Ein Objekt, dessen Attributwert gesetzt werden soll. ja
attribute Zeichenkette Ein Attribute-Literal, welches das zu setzenden Attribut definiert. ja
value Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Der neue Wert des Attributs. ja

Beispiele

	{
   a = new(`my.module:MyClass`);
   $a.set(`my.module:MyClass#name`, "newName");
}
