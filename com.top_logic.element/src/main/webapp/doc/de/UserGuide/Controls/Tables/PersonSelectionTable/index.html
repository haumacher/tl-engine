In Personenauswahltabellen kann man Personen, die im System angelegt sind, Aufgaben und / oder Bereichen zuordnen.   Wird nach der Zuordnung eine Person in dieser Sicht selektiert, wechselt die Anwendung in die Ansicht Personenadministration.

Personenauswahltabelle

Wechselt der Benutzer in den Bearbeiten-Modus, können Personen der Tabelle hinzugefügt, oder entfernt werden.

Die Personenauswahl kann durch das Symbol   geöffnet werden. Der Benutzer kann einen oder mehrere Personen auswählen, die er der Tabelle gerne hinzufügen möchte.

Bereits in der Tabelle enthaltene Personen werden in der Auswahl grau hinterlegt.

Um Personen aus der Auswahltabelle zu entfernen kann der Benutzer die Personenauswahl wieder öffen   und nun den zu entfernenden Datensatz auswählen. Die graue Hinterlegung des Datensatzes wird wieder weiß und mit Übernahme der Änderungen wird der Datensatz dann auch aus der Personenauswahltabelle entfernt.

Es gibt noch eine weitere Möglichkeit Datensätze aus der Personenauswahltabelle zu entfernen.

Aktiviert der Bentuzer den Bearbeitenmodus ist das Symbol "Entfernen" eingeblendet, aber inaktiv . Wird nun ein Datensatz in der Personenauswahltabelle ausgewählt, wird das "Entfernen" Symbol aktiviert und der Nutzer kann den Datensatz mit Klick auf das Symbol aus der Tabelle entfernen.
