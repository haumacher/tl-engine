Syntax

	$e1.isDateEqual($e2)

Beschreibung

Prüft, ob das Datum e1 gleich dem Datum e2 ist. Dabei werden nur ganze Tage verglichen und keine Uhrzeiten berücksichtigt.

Ist äquivalent zu

$e1.day() == $e2.day()

Parameter

Name Typ Beschreibung Pflicht Default
e1 Fachobjekt Datum, das mit e2 verglichen werden soll. ja
e2 Fachbjekt Datum, das mit e1 verglichen werden soll. ja

Rückgabewert

Typ: Wahrheitswert

true, wenn das Ergebnis der Auswertung besagt, dass die beiden Daten auf den Tag genau identisch sind.

Beispiele

Einfaches Datum

	date(2021, 10, 5).isDateEqual(date(2021, 10, 5))

Ausgabe: true

Datum mit Uhrzeit

	dateTime(2021, 10, 5, 12, 24, 55).isDateEqual(dateTime(2021, 10, 5, 15, 24, 23))

Ausgabe: true

Da nur auf den Tag genau verglichen wird und die Tageszeit nicht beachtet wird, wird auch hier true zurückgegeben.
