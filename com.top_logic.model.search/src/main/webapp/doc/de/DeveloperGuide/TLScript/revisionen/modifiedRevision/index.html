​Syntax

	$object.modifiedRevision()

Beschreibung

Ermittelt die Revision in der ein Fachobjekt das letzte mal geändert wurde.

Parameter

Name Typ Beschreibung Pflicht Default
object Fachobjekt Das Fachobjekt für das die Änderungsrevision ermittelt werden soll ja

Rückgabewert

Typ: Revision

Die Revision des Fachobjektes in der das Fachobjekt das letzte mal geändert wurde.
