Wie in Prozess-Daten-Verknüpfung beschrieben, repräsentiert eine Kollaboration eine prozessgesteuerte Instanz des Datenmodells der Anwendung. Die Eigenschaften der Kollaboration sind such Klick auf den Kopfbereich der Kollaboration zu erreichen.

Der "Modell-Typ" stellt die Beziehung zwischen dem Prozess und dem Datenmodell der Anwendung her. Beim Starten einer Kollaboration, wird ein Objekt dieses Typs angelegt und steht den einzelnen Prozesschritten als Modell zur Verfügung. Jede Aufgabe in einer Kollaboration bezieht sich auf dieses Prozessdatenobjekt.

Technisch muss ein Modell-Typ von dem eingebauten Typ "Prozess-Instanz" abgeleitet werden, um als Prozessdatenobjekt fungieren zu können.
