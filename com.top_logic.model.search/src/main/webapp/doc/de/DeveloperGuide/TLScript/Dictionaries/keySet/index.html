Syntax

	$dict.keySet()

Beschreibung

Liefert die Schlüsselmenge eines Wörterbuchs. Die Menge enthält keine doppelten Schlüssel, da Schlüssel unique sein müssen und daher nur einmal in Wörterbuch vorkommen.

Parameter

Name Typ Beschreibung Pflicht Default
dict Fachobjekt Ein Wörterbuch, dessen Schlüssel ausgegeben werden sollen. ja

Rückgabewert

Typ: Menge

Eine Menge, die alle Schlüssel der Wörterbuches dict beinhaltet.

Beispiele

Einfaches Wörterbuch

	{
  dict = {
    "all": "alle",
    "some": "manche",
    "none": "keine"
  };

  $dict.keySet();
}

Ausgabe: Eine Menge mit den Werten ["all", "some", "none"].
