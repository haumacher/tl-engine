Syntax

	$calendar.withWeekOfMonth($week)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Woche des Monats auf week gesetzt wird. Durch das Verändern der Woche können sich auch andere Werte wie zum Beispiel der Tag ändern.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Woche des Monats geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
week Zahl Wert für die neue Monatswoche, der für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Monatswoche.

Beispiele

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withWeekOfMonth(15).weekOfMonth()

Ausgabe: 15
