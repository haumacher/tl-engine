In den Rollenprofilen werden je Berechtigungsstruktur die Berechtigungen auf Sichten eingestellt. Dabei werden je Sicht die einzelnen Bearbeitungsarten (READ, WRITE, DELETE, EXPORT usw.) für ein Rolle festgelegt. Damit ist es sehr granular möglich, die Berechtigungen je Rolle festzulegen.

Hinter den mit Häkchen zu versehenden Berechtigungen steht ein "i", das mit MouseOver die genaue Berechtigung, die sich hinter dem Häkchenfeld befindet, beschreibt.
