Baumstrukturen finden sich häufig dort, wo Hierarchien oder Abhängigkeiten verdeutlicht werden sollen.

Über das Burger-Menü der Baumstruktur lassen sich alle Unterknotenpunkte auf- bzw. zuklappen (siehe Abbildung 1).

Es lassen sich jedoch auch nur einzelne Knotenpunkte erweitern bzw. minimieren (siehe Abbildung 2).

Die Sicht in Baumstrukturen ist meist zweigeteilt. Auf der linken Seite wird der Baum mit seinen Unterknotenpunkten dargestellt, auf der rechten Seite befindet sich die Detailansicht. Durch Anklicken eines Baumelements werden die Eigenschaften des Elements in der Detailansicht angezeigt. Hier können vom Benutzer auch Änderungen vorgenommen werden.
