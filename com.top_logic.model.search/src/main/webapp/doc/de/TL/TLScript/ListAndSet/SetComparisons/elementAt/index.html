Syntax

	$set.elementAt($index)

$set[$index]

Beschreibung

Gibt das Element der Liste set an der Stelle index zurück.

Parameter

Name Typ Beschreibung Pflicht Default
set Menge Eine Menge, aus der ein Element zurückgegeben werden soll. ja
index Zahl Der Index der Liste set, von dem das Element zurückgegeben werden soll. Die Zählung beginnt bei 0 und der gewählte Index darf nur zwischen 0 und $set.size() -1 liegen. ja

Rückgabewert

Typ: Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge

Das Element aus der Liste set an der Position index.

Beispiele

Gültiger Index

	list("hello", "how", "are", "you").elementAt(3)

Ausgabe: you

Ungültiger Index

	list("hello", "how", "are", "you").elementAt(5)

Ausgabe: null

Index ohne elementAt

	list("hello", "how", "are", "you")[2]

Ausgabe: are
