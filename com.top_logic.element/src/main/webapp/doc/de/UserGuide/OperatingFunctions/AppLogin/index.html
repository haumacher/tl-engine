Eine Anmeldung in TopLogic ist nur möglich, falls der Benutzer bereits in der App angelegt und berechtigt ist.

Nach Eingabe von Name und Passwort im Anmelde-Bildschirm wird die TopLogic-Applikation angezeigt. In diesem Bedienkonzept wird grundsätzlich das Theme "Dezent (Sidebar)" verwandt (siehe Persönliche Einstellungen).
