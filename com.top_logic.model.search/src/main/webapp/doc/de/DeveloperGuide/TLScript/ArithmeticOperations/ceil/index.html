Syntax

	ceil($number)

Beschreibung

Rundet die Fließkommazahl number auf die nächstgrößere ganze Zahl auf.

Parameter

Name Typ Beschreibung Pflicht Default
number Zahl Fließkommazahl, die aufgerundet werden soll ja

Rückgabewert

Typ: Zahl

Die nächstegrößere ganze Zahl von number.

Beispiele

Kommazahl

	ceil(3.14)

Ausgabe: 4

Ganze Zahl

	ceil(3)

Ausgabe: 3

Ganze Zahlen bleiben unverändert.
