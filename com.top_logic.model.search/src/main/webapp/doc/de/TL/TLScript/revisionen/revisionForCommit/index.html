Syntax

	revisionForCommit($commit)

Beschreibung

Bestimmt die Revision die zu einem gegebenen Commit gehört.

Parameter

Name Typ Beschreibung Pflicht Default
commit Zahl Die Commitnummer der Revision die bestimmt werden soll. ja

Rückgabewert

Typ: Revision

Die Revision zu einer gegebenen Commitnummer.
