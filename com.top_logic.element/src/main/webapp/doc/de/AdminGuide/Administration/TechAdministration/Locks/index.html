Über Sperren wird verhindert, dass unterschiedliche Nutzer im System versuchen, dieselbe Information gleichzeitig zu bearbeiten. Ohne ein  Sperrkonzept würden sich Änderungen gegenseitig überschreiben.

Sperren von Nutzern können durch den Administrator im System eingesehen und notfalls gewaltsam freigegeben werden. Im normalen Ablauf dürfte dies nicht notwendig sein, da Sperren automatisch freigegeben werden, auch wenn der betreffende Nutzer seinen Bearbeitungsvorgang nicht regulär beendet, sondern einfach seinen Browser schließt. Sollte dennoch eine Sperre nicht automatisch aufgeräumt werden, kann die administrative Sicht "Sperren" verwendet werden, um Sperren vorzeitig (vor ihrem normalen Ablaufdatum) freizugeben.
