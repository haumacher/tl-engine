Firewall Bezüglich Firewalls zwischen Client und Webserver ist zu beachten, dass spezielle Filterungen von Popups etc. wie sie in einem Proxy vorgenommen werden können, NICHT zulässig sind.
Darüber hinaus muss im Hinblick auf Firewall-Einstellungen gewährleistet werden, dass der Applikationsserver auf angebundene Datenhaltungssysteme zugreifen kann (z.B. Directory Server, etc.).
