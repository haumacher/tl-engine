In dieser Sicht werden die fachlichen Rollen der Anwendung festgelegt. Die Rechtevergabe der Rollen erfolgt im Reiter "Rollenprofile".

Grundsätzlich wird die Berechtigungsstuktur "Zentrale Berechtigungen" angelegt. Alle Rollen in dieser Struktur erhalten den Präfix "SecurityStructure.". Der Name der Rolle wird diesem Präfix hinten angestellt.

Sind für bestimmte Strukturen innerhalb der Applikation spezifische Rechtevergaben notwendig, dann muss dafür eine zusätzliche Berechtigungsstruktur angelegt werden. Für jede Berechtigungsstruktur wird ein eigenständiger Präfix vergeben.
