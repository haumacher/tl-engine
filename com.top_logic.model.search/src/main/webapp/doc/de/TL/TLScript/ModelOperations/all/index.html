Syntax

	all(`$class-literal`)

Beschreibung

Ruft alle Instanzen der gegebenen Klasse ab.

Parameter

Name Typ Beschreibung Pflicht Default
class-literal Zeichenkette Die Klasse, deren Instanzen abgerufen werden sollen. ja

Rückgabewert

Typ: Menge

Eine Menge mit allen Instanzen des Klasse.

Beispiele

Alle Instanzen einer Klasse​

	`my.module:MyClass`

Ausgabe: Eine Menge mit allen Instanzen der Klasse MyClass.
