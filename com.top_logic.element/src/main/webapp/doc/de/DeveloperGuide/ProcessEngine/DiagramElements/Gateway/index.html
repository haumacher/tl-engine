Eine Entscheidung im Prozess kann entweder manuell oder automatisch sein. Eine manuelle Entscheidung wird dem Nutzer der davorliegenden Aufgabe beim Abschluss der Aufgabe zur Auswahl gestellt.

In obigem Beispiel dann der Nutzer, welcher die Aufgabe "Decide" ausführt, beim Abschluss der Aufgabe entscheiden, ob abgelehnt oder angenommen wird.

Bei einer automatischen Entscheidung müssen für die ausgehenden Kanten der Entscheidung Regeln hinterlegt werden. In diesem Fall entscheidet das System nach Abschluss der vorherigen Aufgabe automatisch welcher Pfad im Anschluss gegangen wird. Hierfür wird derjenige Pfad gewählt, dessen Regel wahr liefert.
