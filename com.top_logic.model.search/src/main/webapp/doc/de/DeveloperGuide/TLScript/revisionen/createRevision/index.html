Syntax

	$object.createRevision()

Beschreibung

Ermittelt die Revision in der ein Fachobjekt angelegt wurde.

Parameter

Name Typ Beschreibung Pflicht Default
object Fachobjekt Das Fachobjekt für das die Erstellrevision ermittelt werden soll ja

Rückgabewert

Typ: Revision

Die Revision des Fachobjektes in der das Fachobjekt angelegt wurde.

Beispiele

Beispiel 1

	$object.createRevision()

Ausgabe: r42

Wenn das gegebene Fachobjekt object in der 47. Revision angelegt wurde.
