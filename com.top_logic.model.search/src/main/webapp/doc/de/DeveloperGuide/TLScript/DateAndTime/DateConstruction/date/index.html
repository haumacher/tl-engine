Syntax

	date($year, $month, $day)

Beschreibung

Konstruiert einen Datumswert, der einen Tag eines Jahres darstellt (im Systemkalender). Kompatibel mit tl.core:Date Werten.

Dieser Wert kann beispielsweise in Attribute vom Typ tl.core:Date gespeichert, oder mit toSystemCalendar weiter untersucht werden.

Parameter

Name Typ Beschreibung Pflicht Default
year Zahl Jahr des Datums ja
month Zahl Monat des Datums. Die Zählung beginnt bei 0 (Januar = 0, Februar = 1 etc.) ja
day Zahl Tag des Datums. Die Zählung beginnt bei 1 (1 = erster Tag des Monats) ja

Rückgabewert

Typ: Fachobjekt

Einen Datumswert für den angegebenen Tag des Jahres im Systemkalender.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

Unformatiertes Datum

	date(2012, 11, 31)

Ausgabe: 31.12.2012

Formatiertes Datum

	dateFormat("dd-MM-yyy").format(date(2012, 11, 31))

Ausgabe: 30-12-2012
