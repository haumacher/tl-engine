TopLogic – the automated app engine ist eine JEE Applikation und benötigt zum Betrieb einenApplikationscontainer und eine Datenbank. Clientseitig wird ein Standard Browser (JavaScript aktiviert) benötigt.

Applikationsserver

Der Betrieb mehrerer paralleler Applikationsserver (hinter einem Load Balancer, session-sticky) gegen dieselbe Applikationsdatenbank wird unterstützt (optional).
