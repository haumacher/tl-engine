Syntax

	$calendar.withHour($hour)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Stunde auf hour gesetzt wird. Durch das Verändern der Kalenderwoche können sich auch andere Werte wie zum Beispiel die Tageszeit (a.m. oder p.m.) verändern.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Stunde geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
hour Zahl Wert für die neue Stunde, die für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Stunde.

Beispiele

	dateTime(2012, 8, 15, 5, 55, 24, 45).toUserCalendar().withHour(22).hour()

Ausgabe: 22
