Syntax

	$calendar.withMillisecond($millisecond)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Milliesekunde auf millisecond gesetzt wird.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Milliesekunde geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
millisecond Zahl Wert für die neue Milliesekunde, die für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Millisekunde.

Beispiele

	dateTime(2012, 8, 15, 5, 55, 24, 45).toUserCalendar().withMillisecond(22).millisecond()

Ausgabe: 22
