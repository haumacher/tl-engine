Syntax

	$calendar.withSecondAdded($value)

Description

Creates a new calendar from the given calendar, adding value to the second. The addition of the value can have an effect on other values, for example the minute.

Parameters

Name Type Description Mandatory Default
calendar Business object The calendar whose second is to be changed. It can be a system or user calendar. yes
value Number Value to be added to the current second of the calendar. yes

Return value

Type: Business object

A calendar with the changed value for the second.

Examples

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withSecondAdded(2).second()

Output: 26
