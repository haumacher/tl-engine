Zentraler Bestandteil in TopLogic ist die Anzeige der Daten in einer Tabelle. Um das angezeigte Tabellenergebnis besser bearbeitbar zu machen, kann innerhalb jeder Tabellenspalte individuell gefiltert werden. Dazu gibt es grundsätzlich die Möglichkeit in jedem Spaltenkopf der Tabelle zu filtern. Wurde eine Tabellespalte gefiltert wird das Filtersymbol dieser Tabellenspalte rot . Gleichzeitig wird auch im Tabellkopf das Icon (1) rot angezeigt.

Um eine Filterung über die ganze Tabelle vorzunehmen wird im Tabellenkopf das Icon (2) angeklickt. Durch Eingabe eines Textes (3) werden alle Tabellenspalten nach diesem durchsucht und das Ergebnis angezeigt. Sollen alle Zeilen die den eingegebenen Text nicht enthalten, angezeigt werden, so ist die Auswahl "Filterergebnis invertieren" (4) anzukreuzen.

Um die Auswahl der Filtereinstellungen zu löschen wird der Button "Rücksetzen" gedrückt. 

Durch Drücken von "Alle anzeigen" wird der Filter dieser Spalte aufgehoben und das entsprechende Ergebnis in der Tabelle angezeigt.
