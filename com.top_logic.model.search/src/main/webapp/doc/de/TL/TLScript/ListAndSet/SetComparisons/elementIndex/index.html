Syntax

	$set.elementIndex($element)

Beschreibung

Bestimmt die Position eines Elements in einer Liste zurück, wobei die Zählung bei 0 beginnt. Falls das Element mehrfach enthalten ist, wird die Position des ersten Vorkommens zurückgegeben.

Parameter

Name Typ Beschreibung Pflicht Default
set Menge Eine Menge, in der die Position eines Elements bestimmt werden soll. ja
element Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Das Element dessen Position bestimmt werden soll. ja

Rückgabewert

Typ: Zahl

Den Index der Position des Elements in der Menge. Falls das Element mehrfach vorkommt wird der Index des ersten Vorkommens zurückgegeben. Falls das Element gar nicht enthalten ist, wird -1 zurückgegeben.

Beispiele

Element befindet sich 1 Mal in der Menge

	list(1, 2, 3, 4).elementIndex(2)

Ausgabe: 1

Element befindet sich merhfach in der Menge

	list(1, 2, 2, 2, 3, 4).elementIndex(2)

Ausgabe: 1

Element befindet sich nicht in der Menge

	list(1, 2, 2, 2, 3, 4).elementIndex(8)

Ausgabe: -1
