Zur Vereinfachung und Sicherheit, lassen sich die Berechtigungsprofile der TopLogic Rollen in eine XML-Datei (z.B.: "security.xml") exportieren und bei Bedarf wieder importieren (nur XML-Dateien). Zur Dokumentation können die Rollen auch in eine Datei im Excel-Format exportiert werden, allerdings läßt sich diese Format nicht mehr importieren.

Über den Reiter Administration : Berechtigungen : Rollenprofile gelangt man zur Ansicht der Rollenprofile.
