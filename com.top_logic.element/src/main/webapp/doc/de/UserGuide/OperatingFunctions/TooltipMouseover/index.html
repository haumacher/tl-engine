Um für Buttons, Tabellenspalten, Tabelleninhalte etc. schnell eine Info zu bekommen, wird in TopLogic mit Tooltips (MouseOver) gearbeitet.Der Tooltip erscheint, wenn der Mauszeiger eine kurze Zeit unbewegt über dem entsprechenden Element verweilt. In der Abbildung steht der Mauszeiger auf und als Tooltip wird "Suchergebnisse aktualisieren" angezeigt.

Tooltips zeigen ergänzende Informationen zu dem zugehörigen Element, die ansonsten nicht sichtbar sind. Tooltips können auch den Text anzeigen, den das Element selbst auch enthält. Beispielsweise bei einer Tabellenzelle, die mehr Text enthält als in dem zur Verfügung stehenden Platz angezeigt werden kann.
