Für die Nutzer-Authentifizierung stehen verschiedene Verfahren zur Verfügung. Die Nutzer-Authentifizierung kann entweder an ein extenes System (LDAP, OpenID) delegiert werden, oder über eine eingebaute Nutzerdatenbank erfolgen.

Wird nichts konfiguriert, werden Anmeldeinformationen in der eingebauten   Nutzerdatenbank gespeichert. Insbesondere der Initial-Nutzer und Super-Administrator (root) einer Anwendung hat seine Anmeldeinformationen immer in der eingebauten Nutzerdatenbank hinterlegt.
