Die Sicht demonstriert die Verwendung von Drag&Drop zwischen Komponenten. Auf der rechten Seite kann eine Template-Struktur administriert werden und daraus können Elemente in den Fabrikplan auf der linken Seite mit Drag&Drop eingefügt werden. Die Elemente des Fabrikplans können ebenfalls über Drag&Drop umgeordnet werden. Dabei erhält der Nutzer entsprechendes Feedback, wo welche Elemente eingefügt werden können.

In dem zugehörigen Beispielmodell können bestehen Fabriken aus Stockwerken und Stockwerke aus Arbeitsplätzen. Bei der Drag&Drop-Operation können entsprechend Stockwerke nur in Fabriken und Arbeitsplätze nur innerhalb von Stockwerken plaziert werden.

Import von Bibliothekselementen

Im Burger-Menü der Template-Bibliothek befindet sich ein Import-Kommando, mit dem aus einer XML-Struktur Bibliothekselemente importiert werden können. Diese Funktionalität demonstriert den konfigurierbaren XML-Import. Wenn man die folgende Struktur in eine Datei speichert, können hiermit über den Import Bibliothekselemente in der Template-Bibliothek angelegt werden:

<items>
	<factory name="F1">
		<floor name="Ebene 0">
			<location name="Raum1">
				<entry name="Robot 1"/>
			</location>
		</floor>
	</factory>

	<floor name="Ebene">
		<location name="Raum A">
			<Entry name="Robot 1"/>
		</location>
		<location name="Raum B">
			<Entry name="Robot 2"/>
		</location>
	</floor>
</items>
