Syntax

	$string.length()

Beschreibung

Gibt die Anzahl der Zeichen in der Zeichenkette string an.

Parameter

Name Typ Beschreibung Pflicht Default
string Zeichenkette Zeichenkette, dessen Länge berechnet werden soll. ja

Rückgabewert

Typ: Zahl

Die Anzahl der Zeichen in string.

Beispiele

	length("Hello world!")

Ausgabe: 12
