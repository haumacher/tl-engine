Der Wartungsmodus dient dem Administrator dazu, Änderungen in der Anwendung durchzuführen ohne dass konkurrierende Zugriffe auf die Anwendung und die Datenbank durch andere Benutzer stattfinden.

Durch aktivieren des Wartungsmodus

  • können sich Benutzer ohne Administratorrechte nicht mehr anmelden
  • werden angemeldete Benutzer informiert, dass der Wartungsmodus in einer festzulegenden Zeitspanne aktiviert wird
  • werden angemeldete Benutzer (ohne Administratorrechte) nach dieser Zeitspanne abgemeldet

Nachdem der Button "Wartungsmodus aktivieren" gedrückt wurde, erscheint ein Dialog-Fenster, in dem die Zeitspanne bis zum aktivieren des Wartungsmodus eingegeben werden kann.

Ohne Eingabe eines Wertes wird der Wartungsmodus sofort aktiv.

Durch Drücken von "Wartungsmodus aktivieren" erscheint für die festgelegte Zeitsspanne die Meldung "Das System begibt sich demnächst in den Wartungsmodus". Dieser Zustand läßt sich durch Drücken des Buttons "Wartungsmodus abbrechen" beenden, um in den normalen Systemmodus zurückzuwechseln.

Befindet sich das System im Wartungsmodus erscheint die Meldung "Das System befindet sich zur Zeit im Wartungsmodus". Dieser Zustand läßt sich durch Drücken des Buttons "Wartungsmodus abschaltenn" beenden, um in den normalen Systemmodus zurückzuwechseln.
