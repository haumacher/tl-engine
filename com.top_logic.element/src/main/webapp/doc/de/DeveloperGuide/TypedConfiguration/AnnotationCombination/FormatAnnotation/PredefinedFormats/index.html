Es gibt viele vordefinierte Formate für Java-Standard-Typen wie z.B.

  • das Format ​HexEncodedPaint für Color;
  • ​MillisFormat, um eine Angabe "5min 30s" in einen long-Wert in Millisekunden einzulesen.
  • das Format CommaSeparatedStrings, um eine komma-separierte String-Liste 
    public interface Config extends ConfigurationItem {
  @Format(CommaSeparatedStrings.class)
  List<String> getMyListValues();
}

    aus

    <config my-list-values="bli,bla,blub" />

    einzulesen. Ähnlich liefert CommaSeparatedStringSet  ein Set statt einer List. Dafür muss natürlich auch die Eigenschaft als Set<String> deklariert sein.
