Syntax

	now()

Beschreibung

Konstruiert einen Datumswert, der den aktuellen Zeitpunkt darstellt (im Benutzerkalender). Kompatibel mit tl.core:DateTime Werten.

Rückgabewert

Typ: Fachobjekt

Einen Datumswert, der dem aktuellen Zeitpunkt gemäß des Benutzerkalenders entspricht.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

Unformatiertes Datum

	now()

Ausgabe: 06.10.2021

Formatiertes Datum

	dateFormat("dd-MM-yyy hh:mm:ss").format(now())

Ausgabe: 06-10-2021 02:33:35
