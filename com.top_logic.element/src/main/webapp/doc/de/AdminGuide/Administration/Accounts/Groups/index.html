Hier werden die vorhandenen fachlichen Gruppen verwaltet. Auf der linken Seite werden die Gruppen aufgeführt. Wird eine Gruppe angeklickt werden auf der rechten Seite  des Bildschirms die zugeordneten Benutzer in Tabellenform dargestellt.

Im Bearbeitungsmodus lassen sich ausser den Gruppenstammdaten auch Benutzer den fachlichen Gruppen hinzufügen bzw. abwählen.

Die dargestellten fachlichen Gruppen entsprechen den Gruppen unter "Administration : Berechtigungen : Gruppenrechte". Dort werden die Berechtigungen der Gruppen festgelegt.
