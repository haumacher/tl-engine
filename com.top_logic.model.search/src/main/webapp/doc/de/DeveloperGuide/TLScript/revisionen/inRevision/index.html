Syntax

	$object.inRevision($revision)

Beschreibung

Löst das Fachobjekt in der gegeben Revision auf.

Parameter

Name Typ Beschreibung Pflicht Default
object Fachobjekt Fachobjekt das in der gegebenen Revision aufgelöst werden soll ja
revision Revision oder Zahl Die Revision oder Commitnummer in der das Fachobjekt angezeigt werden soll. Die Commitnummer muss >=0 sein und existieren. ja

Rückgabewert

Typ: Fachobjekt

Das Fachobjekt in der gegebenen Revision oder null, wenn das Fachobjekt in der Revision nicht existert.
