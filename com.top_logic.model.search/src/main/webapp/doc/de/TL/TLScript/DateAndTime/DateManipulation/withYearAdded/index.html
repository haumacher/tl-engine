Syntax

	$calendar.withYearAdded($value)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei das Jahr um value erhöht wird. Die Addition des Wertes kann Auswirkungen auf andere Werte haben, zum Beispiel auf den Tag der Woche.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Tag geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
value Zahl Wert, der auf das aktuelle Jahr des Kalenders addiert werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für das Jahr.

Beispiele

	date(2012, 8, 10).toUserCalendar().withYearAdded(2).year()

Ausgabe: 2014
