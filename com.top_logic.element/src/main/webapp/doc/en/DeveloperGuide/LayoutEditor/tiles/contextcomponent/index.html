There are several types of components that can be selected as context components:

  • Tables
  • Tree tables
  • Trees
  • Tile display of objects

For all components the displayed elements are defined by a TL-Script expression.

When an element is selected, this object is not displayed separately, in contrast to a normal component. Instead, it is descended further in the tile hierarchy, so that further components can be configured, which the selected element can now use, e.g. as a model.
