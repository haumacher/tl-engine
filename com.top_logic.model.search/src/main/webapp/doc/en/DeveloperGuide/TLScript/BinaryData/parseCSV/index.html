Syntax

	$input.parseCSV()

Description

Interprets the input as CSV formatted text.

Parameters

name Type Type Description Mandatory Default
Input Binary value Text in CSV format Yes -

Return value

Type: List

Creates a list for each line in the input and then combines these lists into one list.

Example:

{
    toParse = binary(
        name: "test.csv",
        data: "date; name; number
        15.07.2024; test; 5",
        contentType: "text/csv",
        encoding: "utf-8"
    );
    result = $toParse.parseCSV();
    $result[0]
}

Output: ["date", "name", "number"]

Calling parseCSV() returns the list [["date", "name", "number"], ["15.07.2024", "test", "5"]].
