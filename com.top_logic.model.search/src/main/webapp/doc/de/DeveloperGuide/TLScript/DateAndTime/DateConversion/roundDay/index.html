Syntax

	roundDay($timestamp)

Beschreibung

Wandelt den Zeitpunkt timestamp in ein Datum um.

Parameter

Name Typ Beschreibung Pflicht Default
timestamp Fachobjekt EIn Datumswert, der einen Zeitpunkt darstellt. ja

Rückgabewert

Typ: Fachobjekt

Ein Datumswert, der den angegebenen Zeitpunkt repräsentiert, wobei Angaben für die Zeit (Stunden, Minuten etc.) zurückgesetzt werden.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

	dateFormat("dd-MM-yy hh:mm:ss").format(roundDay(now()))

Ausgabe: 07-10-21 05:00:00
