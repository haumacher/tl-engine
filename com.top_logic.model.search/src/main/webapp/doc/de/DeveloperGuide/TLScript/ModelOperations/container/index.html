Syntax

$object.container()

Beschreibung

Liefert den Container des Objektes object. Der Container eines Objektes ist dasjenige Objekt x, das eine Referenz auf object in einer Compositions-Referenz hält.

Parameter

Name Typ Beschreibung Pflicht Default
object Fachobjekt Das Objekt, dessen Container ermittelte werden soll. ja

Rückgabewert

Typ: Fachobjekt

Den Container von object. Hat object keinen Container, liefert die Funktion null.

Beispiele

	{
   a = new(`my.module:MyClass`);
   b = new(`my.module:MyClass`);
   $a.add(`my.module:MyClass#children`, $b);
   $b.container();
}

Ausgabe: Das Objekt a.

b wird hier in das Referenzattribut children von a eingetragen. a ist der Container von b.
