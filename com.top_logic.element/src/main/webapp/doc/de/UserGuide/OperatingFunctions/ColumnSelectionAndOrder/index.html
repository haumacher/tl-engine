Zentraler Bestandteil in TopLogic ist die Anzeige der Daten in einer Tabelle. Um das angezeigte Tabellenergebnis besser bearbeitbar zu machen kann der angezeigte Inhalt der Tabelle durch hinzufügen und abwählen von Spalten beliebig angepasst werden.

Damit die Tabellenspalten den Benutzeranforderungen genügen, kann über das Burger-Menü (1) der Unterpunkt "Spaltenauswahl und -reihenfolge" ausgewählt werden. Im angezeigten Popup-Window kann der Tabelle jedes angezeigte Attribut hinzugefügt oder abgewählt (2). Durch Drücken von "Zurücksetzen" wird der ursprüngliche Zustand der Tabellenspalten wieder hergestellt.
