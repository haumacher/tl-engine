Syntax

	$calendar.withWeekAdded($value)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Kalenderwoche um value erhöht wird. Die Addition des Wertes kann Auswirkungen auf andere Werte haben, zum Beispiel auf den Tag.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Kalenderwoche geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
value Zahl Wert um den die Kalenderwoche erhöht werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Kalenderwoche.

Beispiele

Wochennummer nach Anpassung

	dateTime(2012, 8, 15, 7, 55, 24, 13).toUserCalendar().withWeekAdded(5).week()

Ausgabe: 42

Tag nach Anpassung

	dateTime(2012, 8, 15, 7, 55, 24, 13).toUserCalendar().withWeekAdded(5).day()

Ausgabe: 20

Die Anpassung von week ändert auch andere Werte wie den Tag.
