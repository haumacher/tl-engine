Listen und Mengen in TL-Script sind Ansammlungen von mehreren Einzelwerten. Sie unterscheiden sich nur darin, dass eine Liste explizit geordnet ist und ein und dasselbe Element mehrfach enthalten kann, eine Menge aber nicht. Ansonsten können Listen und Mengen austauschbar verwendet werden. Eine explizite Umwandlung einer Liste in eine Menge über toSet kann dann sinnvoll sein, wenn der Zugriffe auf eine Modell-Referenz eine Mehrfach-Liste liefert, aber Duplikate explizit unterdrückt werden sollen.

Genauso wie Variablen können werde Listen, noch Mengen nachträglich verändert werden (z.B. hinzufügen oder entfernen von Elementen). Stattdessen müssen neue Listen oder Mengen generiert werden, zum Biepsiel mit der Verkettung.
