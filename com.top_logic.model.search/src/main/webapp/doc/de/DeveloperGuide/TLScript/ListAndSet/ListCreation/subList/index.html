Syntax

	$set.subList($startIndex)

$set.subList($startIndex, $stopIndex)

Beschreibung

Gibt einen Teil der gegebenen Liste zurück, der durch die Indizes festgelegt wird.

Parameter

Name Typ Beschreibung Pflicht Default
set Aufzählung Eine Liste, von der eine Teilliste entnommen werden soll. ja
startIndex Zahl Der Index von set, an dem die Teilliste beginnen soll (einschließlich). Das erste Element von set ist 0. Das Element mit dem angegebenen startIndex ist in der Teilliste mit inbegriffen. ja
stopIndex Zahl Der Index von set, an dem die Teilliste enden soll (ausschließlich). Das erste Element von set ist 0. Das Element mit dem angegebenen stopIndex ist in der Teilliste nicht mit inbegriffen. nein Die Teilliste enthält alle Elemente von startIndex bis zum Ende von list. Äquivalent zu $set.subList($startIndex, $set.size()).

Rückgabewert

Typ: Mengte

Eine Teilliste der angegebenen Menge set.

Beispiele

Teilliste ohne stopIndex

	list(1, 2, 3, 4, 5).subList(2)

Ausgabe: Eine Menge mit den Elementen [3, 4, 5].

An Index 2 steht die Zahl 3. Von dort aus werden alle Elemente mit in die Teilliste aufgenommen.

Teilliste mit stopIndex

	list(1, 2, 3, 4, 5).subList(2, 4)

Ausgabe: Eine Menge mit den Elementen [3, 4].

An Index 2 steht die Zahl 3 und Index 4 die Zahl 5. Der stopIndex wird nicht in die Teilliste mit einbezogen.
