Der Reiter "Organisationseinheiten" bildet die Strukturen des Unternehmens ab.

Über den Button "Neu" lässt sich der Anlagedialog zum erstellen einer neuen Organisationseinheit öffnen.

Nachdem der Name der Organisationseinheit eingegeben und angelegt wurde, erscheint die Organisationseinheit an der Stelle im Baum, die man vorher ausgewählt hatte.

Die ID der Organisationseinheit wird automatisch vom System vergeben und kann nicht geändert weren. Über das DropDown-Feld "Leiter" kann der Leiter der Organisationseinheit ausgewählt werden.

Mitarbeiter können über die Auswahlfunktion der Tabelle "Mitarbeiter" der Organisationseinheit zugeordnet werden.
