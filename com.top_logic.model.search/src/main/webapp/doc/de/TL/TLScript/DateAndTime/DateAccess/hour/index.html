Syntax

	$calendar.hour()

Beschreibung

Die Stunde des gegebenen Kalenders.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Stunde ermittelt werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja

Rückgabewert

Typ: Zahl

Eine Zahl, die die Stunde des gegebenen Kalenders repräsentiert.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

	dateTime(2012, 8, 5, 8, 55, 23, 16).toUserCalendar().hour()

Ausgabe: 8
