Zweck

  • Aufzeichnen von neuen Testfällen
  • Wartung von bestehenden Testfällen

Installation

  • Die Scripting Gui ist standardmäßig nur im Entwickler-Modus aktiv. (Siehe: enable-debug.xml)
    • Sie ist nicht für Kunden geeignet, da das Scripting-Framework weder von der Usability noch von der Qualität Kunden zugemutet werden kann.

Aufbau

  • Ganz oben gibt es eine Liste mit der man Skripte vom Client oder Server laden kann. Skripte vom Server können nach dem Editieren gespeichert werden.
  • Links befindet sich der Baum der aufgezeichneten Actions
  • In der Mitte befindet sich das XML der im Baum ausgewählten Action

Funktionen

  • Aufzeichnen von Aktionen
    • Nach dem Klick auf "Start" werden Aktionen in der Anwendung aufgezeichnet.
      • Aktionen in der Scripting Gui selber natürlich nicht.
    • Durch den Klick auf "Aktionen hinzufügen" werden diese aufgezeichneten Aktionen in der Gui in den Baum hinzugefügt.
      • Vorher sieht der Benutzer nichts von den aufgezeichneten Aktionen.
  • Ausführen von Aktionen: Eine Action im Baum auswählen, "Skript ausführen" klicken ⇒ Die Action wird mitsamt ihrer Kinder ausgeführt.
  • Lesbare Darstellung von Actions: Rechts oben wird die aktuelle Action inklusive ihrer Kinder in menschenlesbarer Form angezeigt.
  • Editieren von Actions: Wenn eine Action im Baum selektiert ist, erscheint rechts das XML dieser Action.
    • Bei Actions mit Kindern enthält das XML auch diese.
    • Ändert man das XML, wird auch die Action entsprechend geändert.
    • Dadurch kann man Actions umsortieren, indem man deren gemeinsamen Elternknoten editiert.
    • Wird ungültiges XML eingegeben, wird dieses ignoriert. ( Die Action wird nicht verändert.)
      • Daher müssen Sonderzeichen XML-Konform encodiert werden. Zum Beispiel &, < und >.
  • Namen für Actions: Fügt man einer Action ein comment-Attribut hinzu, wird dessen Wert im Baum als Name für die Action verwendet.
  • Actions Debuggen: Wird eine Action mit Kindern ausgeführt und eine der Actions schlägt fehl, wird die fehlgeschlagene Action im Baum selektiert.
