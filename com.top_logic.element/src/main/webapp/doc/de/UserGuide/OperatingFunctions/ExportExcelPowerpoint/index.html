Grundsätzlich kann jede Tabelle in das Excel- bzw. Powerpoint-Format konvertiert und zur Weiterbearbeitung angeboten werden. Aufgerufen wird der Export im Burger-Menü (1) durch die Auswahl "Nach Excel exportieren" (2). Dabei werden die angezeigten Daten der TopLogic-Tabelle mit den Spalten und Zeilen exportiert, die zuvor durch Sortierung und Filterung ausgewählt wurden.
