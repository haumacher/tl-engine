Syntax

	$calendar.withMillisecondAdded($value)

Description

Creates a new calendar from the given calendar, increasing the millisecond by value. The addition of the value can have an effect on other values, for example the second.

Parameters

Name Type Description Mandatory Default
calendar Business object The calendar whose millisecond is to be changed. It can be a system or user calendar. yes
value Number Value to be added to the current millisecond of the calendar. yes

Return value

Type: Business object

A calendar with the changed value for the millisecond.

Examples

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withMillisecondAdded(2).millisecond()

Output: 16
