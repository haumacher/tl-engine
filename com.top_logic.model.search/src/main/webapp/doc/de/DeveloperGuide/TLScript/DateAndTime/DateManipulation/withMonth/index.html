Syntax

	$calendar.withMonth($month)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei der Monat auf month gesetzt wird. Durch das Verändern des Monats können sich auch andere Werte wie zum Beispiel die Kalenderwoche ändern.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Monat geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
month Zahl Wert für den neuen Monat, der für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für den Monat.

Beispiele

	date(2012, 8, 15).toUserCalendar().withMonth(3).month()

Ausgabe: 3
