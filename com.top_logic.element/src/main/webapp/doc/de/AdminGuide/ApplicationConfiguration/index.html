Die Anwendungskonfiguration in TopLogic wird für jede Stage (z.B.: Test, Prelife, Life) in einer Konfigurationsdatei (z.B.: "config-<stage>.xml") gespeichert. Im Deployment ist die abgestimmte Anwendungskonfiguration der Zielumgebung in der Konfigurationsdatei definiert.

Die folgende Liste beschreibt beispielhaft, die in der Konfigurationsdatei spezifizierten Werte:

  • Applikationsverzeichnis
  • Aufruf URL
  • Datenbankanbindung
  • E-Mail Konfiguration
  • Directory Konfiguration
  • Externe Authentifizierung (ISAM Anmeldung)
  • Group DMS Connector
  • etc.  

Die Anwendungskonfiguration wird ausführlich im Administratorhandbuch beschrieben.
