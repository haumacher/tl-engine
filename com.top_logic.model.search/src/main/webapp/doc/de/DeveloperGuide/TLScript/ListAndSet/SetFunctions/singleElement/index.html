Syntax

	$set.singleElement()

Beschreibung

Packt eine einelementige Menge aus, indem das einzige Element als Ergebnis geliefert wird. Ist die Eingabe null, oder die leere Menge, ist das Ergebnis null. Falls die Menge mehrere Elemente enthält, dann wird ein Fehler ausgegeben.

Siehe auch Einpacken.

Parameter

Name Typ Beschreibung Pflicht Default
set Menge Eine Menge oder Liste mit nur einem Element, das ausgepackt werden soll. Mehrelementige Mengen sind nicht erlaubt. ja

Rückgabewert

Typ: Zahl/Zeichenkette/Wahrheitswert/Fachobjekt

Das Element, das in set enthalten ist oder null, falls die Eingabe null oder die Menge leer war.

Beispiele

Einelementige Menge

	list(1).singleElement()

Ausgabe: 1

Leere Menge

	none().singleElement()

Ausgabe: null

Die leere Menge, die durch none() erzeugt wird, hat kein Element zum Auspacken.
