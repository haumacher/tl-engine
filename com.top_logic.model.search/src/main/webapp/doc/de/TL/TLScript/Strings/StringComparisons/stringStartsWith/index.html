Syntax

	$string1.stringStartsWith($string2)

$string1.stringStartsWith($string2, $case-sensitive)

Beschreibung

Prüft ob string1 mit der Zeichenkette string2 beginnt.

Parameter

Name Typ Beschreibung Pflicht Default
string1 Zeichenkette Eine Zeichenkette, für die geprüft wird, ob sie mit string2 beginnt. ja
string2 Zeichenkette Eine Zeichenkette, für die geprüft wird, ob sie am Anfang von string1 steht. ja
case-sensitive Wahrheitswert Bestimmt, ob auch die Groß- und Kleinschreibung berücksichtigt werden soll. nein false - die Groß- und Kleinschreibung wird nicht berücksichtigt.

Rückgabewert

Typ: Wahrheitswert

Ob string1 mit string2 beginnt.

Beispiele

Einfacher Test

stringStartsWith("Hello world", "Hello")

Ausgabe: true

Casesensitiver Test 1

stringStartsWith("Hello world", "Hello", true)

Ausgabe: false

Casesensitiver Test 2

stringStartsWith("Hello world", "hello", true)

Ausgabe: false
