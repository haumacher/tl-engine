Auf jeder Bildschirmseite von TopLogic wird im Kopf der gesamte Pfad, der zur Anzeige dieses Bildschirms führte, als "Doppelpunkt"-separierte Liste angezeigt.

Diese Liste zeigt den Pfad zum aktuellen Element, beispielsweise zu "Rollen prüfen" der Anwendung. Brotkrümel-Navigation soll die Orientierung innerhalb tief verzweigter Elementbäume verbessern, indem sie Links zu übergeordneten Elementen anbieten (z.B.: "Administration").
