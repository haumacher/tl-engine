Zentraler Bestandteil in TopLogic ist die Anzeige der Daten in einer Tabelle. Um das angezeigte Tabellenergebnis besser bearbeitbar zu machen, kann grundsätzlich innerhalb jeder Tabellenspalte individuell gefiltert werden. Dazu gibt es die Möglichkeit im Spaltenkopf die Tabelle zu filtern. Wurde eine Tabellespalte gefiltert wird das Filtersymbol dieser Tabellenspalte rot. Gleichzeitig wird auch im Tabellkopf das Icon rot angezeigt.

Um eine Filterung über die ganze Tabelle vorzunehmen wird im Tabellenkopf das Icon angeklickt. ere Spalten durchzuführen, wird über das Burger-Menü (1) der Unterpunkt Sortierung ausgewählt. Im angezeigten Popup-Window kann dann die Tabelle über jede Spalte sortiert werden (2).
