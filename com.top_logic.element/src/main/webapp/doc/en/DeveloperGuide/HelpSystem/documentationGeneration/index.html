Generating documentation from the application structure

For a newly created application, it is important to provide the user with documentation on which views can be used for which purpose.

For this purpose the command Generate documentation can be used in the view Administration > Development > Help editor.

This command generates documentation templates below the selected page based on the application structure. For each view in which a help ID is set, a page is created. If necessary, parent pages are created to mimic the tab and dialog structure of the application.

This command can be executed multiple times; pages are only created for views that do not yet exist. Pages that have already been created are not changed.
