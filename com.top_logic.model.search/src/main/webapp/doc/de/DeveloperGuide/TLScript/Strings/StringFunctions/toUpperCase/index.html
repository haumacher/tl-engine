Syntax

	$string.toUpperCase()

Beschreibung

Alle Buchstaben der Zeichenkette string werden zu Großbuchstaben umgewandelt.

Parameter

Name Typ Beschreibung Pflicht Default
string Zeichenkett Zeichenkette, dessen Buchstaben umgewandelt werden sollen. ja

Rückgabewert

Typ: Zeichenkette

string in Großbuchstaben.

Beispiele

	toUpperCase("Hello world!")

Ausgabe: HELLO WORLD!
