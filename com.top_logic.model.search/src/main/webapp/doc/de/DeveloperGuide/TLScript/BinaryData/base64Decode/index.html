Syntax

$input.base64Decode()

Beschreibung

Dekodiert die Base64 kodierte Zeichenkette und liefert die zugehörigen Binärdaten.

Parameter

Name Typ Beschreibung Pflicht Default
input Zeichenkette Base64 kodierter Inhalt ja -
name Zeichenkette Name der zu erzeugenden Datei nein "data"
contentType Zeichenkette Content-Type der zu erzeugenden Datei nein Wird aus dem Namen geschlossen, z.B. "text/plain" für "*.txt".

Rückgabewert

Typ: Binärdaten

Der aus der Base64-Kodierung extrahierte Binärwert.

Beispiel

	"TWVpbiBUZXh0".base64Decode(name: "file.txt")

Ausgabe: Eine Text-Datei mit Inhalt "Mein Text".
