Syntax

	$e1 == $e2

$e1.equals($e2)

Description

Boolean expression that evaluates to true if e1 and e2 evaluate to the same value.

Parameter

Name Type Description Mandatory Default
e1 Number/string/boolean/Business object/set Object to be compared with e2. yes
e2 Number/string/boolean/Business object/set Object to be compared with e1. yes

Return value

Type: Boolean

true, if the result of the evaluation says that the two objects are identical.

Examples

Numbers

	1 == 1

Output: true

Equal quantities

	list(1, 2).equals(list(1, 2))

Output: true

Unequal quantities

	list(1, 2).equals(list("one", "two"))

Output: false
