Syntax

	$calendar.withWeek($week)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Kalenderwoche auf week gesetzt wird. Durch das Verändern der Kalenderwoche können sich auch andere Werte wie zum Beispiel der Monat ändern.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Kalenderwoche geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
week Zahl Wert für die neue Kalenderwoche, der für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Kalenderwoche.

Beispiele

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withWeek(15).week()

Ausgabe: 15
