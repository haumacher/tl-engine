Syntax

	currentUser()

Beschreibung

Liefert den aktuell am System angemeldeten Nutzer, in dessen Kontext das laufende Skript ausgeführt wird. Das Ergebnis ist kein Kontakt (Contacts:PersonContact), sondern der tatsächliche Anwendungszugang (tl.legacy.tabletypes:PersonTable).

Rückgabewert

Typ: tl.legacy.tabletypes:PersonTable

Den aktuell angemeldeten Nutzer
