Syntax

	$calendar.withHour12Added($value)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Stunde im 12-Stunden Format um value erhöht wird. Die Addition des Wertes kann Auswirkungen auf andere Werte haben, zum Beispiel auf den Tag.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Stunde geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
value Zahl Wert, der auf die aktuelle Stunde des Kalenders addiert werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Stunde.

Beispiele

hour <= 12

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withHour12Added(2).hour12()

Ausgabe: 9

hour > 12

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withHour12Added(6).hour12()

Ausgabe: 1
