Es können neue Personen dem System hinzugefügt werden. Durch das Anlegen einer neuen Person wird diese nicht automatisch zum Benutzer. Soll eine Person auch Nutzer des Systems sein muss diese in der Sicht "Administration:Zugänge:Benutzerkonten" angelegt oder durch externe Benutzerverwaltungssysteme synchronisiert werden.

Nach dem Anlegen der neuen Person über den Anlagedialog können weitere Personendetails hinzugefügt werden.

Neben den Kontaktinformationen der Person kann man auch Stellvertreter und Vorgesetzter, sowie das Unternehmen auswählen.

 

Im unteren Bereich sind zugeordnete Mitarbeiter abgebildet und die Stellvertreterpositionen.
