In design mode, table bars and also the side bar get a menu button that can be used to customize these components.

Here, new tabs can be added ("Add Tab") and the order of the tabs can be adjusted ("Configure Tabs").

Add Tab

The menu item "Add Tab" can be used to create a new tab with label and optionally an icon in the respective tab bar or side bar.

A new tab on the top level should always be provided with an icon. Since such a tab can be displayed in the sidebar, depending on the theme, and the sidebar only displays the icons when collapsed, a tab without an icon could not be displayed in this situation.

After completing the operation, the new tab can be selected directly:

The desired position in the table bar or side bar can then be adjusted using the "Configure Tabs" menu item.

Fill tab content

The newly created view is still empty. It can then be filled with content. To fill the view with a new component, the plus button in the toolbar can be used.

For the new component to be created, you can select from a list of templates.

The new view will then fill the entire area of the new tab. If several views are to be displayed next to each other in the new tab, the view can be divided into several areas using the menu item "Edit layout", see Customize layout.

Sub-tabs

If another tab bar is to be displayed below a tab, the option "Tabbar" must be selected as the new view. In the dialog that opens, you can then enter names for the sub-tabs to be displayed. The new tab bar created in this way can be supplemented later with further tabs in the same way.
