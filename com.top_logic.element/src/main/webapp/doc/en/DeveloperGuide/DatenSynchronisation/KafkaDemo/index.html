Anleitung zum Starten einer Kafka Demo

Starten von ZooKeeper

In den Debug Configuration in Eclipse findet man im öffnenden Dialog unter Java Application den Eintrag

Start ZooKeeper

um ​ZooKeeper zu starten.

Starten von Kafka

In den Debug Configuration in Eclipse findet man im öffnenden Dialog unter Java Application den Eintrag

Start Kafka

um den ​Kafka Server zu starten.

Starten der Kafka Demo für den Producer und Consumer

Für eine Demo benötigt man einen Client ​Producer, der Narichten verschickt und einen Client ​Consumer, der die Narichten empfangen kann. Deswegen müssen zwei weitere Module gestartet werden.

Starten der Kafka Demo für den Producer

In den Debug Configuration in Eclipse findet man im öffnenden Dialog unter Java Application den Eintrag

Start Start Top Logic Kafka Demo

um die Kafka Demo für den Producer zu starten. Unter ​http://localhost:8080/kafkaDemo/ findet man nun die Demo für den Producer.

Starten der Kafka Demo für den Consumer

In den Debug Configuration in Eclipse findet man im öffnenden Dialog unter Java Application den Eintrag

Start Start Top Logic Kafka Demo (8081)

um die Kafka Demo für den Consumer zu starten. Unter ​http://localhost:8081/kafkaDemo/ findet man nun die Demo für den Consumer.
