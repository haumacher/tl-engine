Die Applikation unterstützt eine automatisierte Hintergrund-Überwachung durch Aufruf einer dafür spezifizierten Seite z.B. “…/App/SystemState.jsp”.

Beim Aufruf dieser Seite werden die gleichen Überprüfungen durchgeführt wie beim Aufruf des Applikationsmonitors. Sind alle Überprüfungen erfolgreich wird ein abgestimmter Text z.B. “System is up and running” ausgegeben.

Dieser Text kann durch Hintergrund-Überwachungstools abgerufen und als Vergleichswert herangezogen werden, um den Systemstatus automatisiert zu überprüfen.

Im Fehlerfall gibt diese Seite anstatt dieser Meldung die Ampelübersicht des Applikationsmonitors aus, so dass ersichtlich ist, welches die problematischen Komponenten sind.
