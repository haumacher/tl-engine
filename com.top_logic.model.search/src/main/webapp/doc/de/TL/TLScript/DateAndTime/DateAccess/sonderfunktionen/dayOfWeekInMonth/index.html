Syntax

	$calendar.dayOfWeekInMonth()

Beschreibung

Der Tag der Woche im Monat im gegebenen Kalender. Damit wird angegeben, der wievielte entsprechende Wochentag des Monats es ist, also zum Beispiel der weivielte Donnerstag.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Tag der Woche im Monat ermittelt werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja

Rückgabewert

Typ: Zahl

Eine Zahl, die den Tag der Woche im Monat des gegebenen Kalenders repräsentiert.

Beispiele

	date(2021, 9, 12).toSystemCalendar().dayOfWeekInMonth()

Ausgabe: 2

Es ist der zweite Dienstag des Monats. Der erste war am 05.10.2021.
