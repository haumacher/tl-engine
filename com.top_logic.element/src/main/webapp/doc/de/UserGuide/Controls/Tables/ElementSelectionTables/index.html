In einigen Tabellen werden Inhalte als Link dargestellt. Klickt der Benutzer auf einen dieser Links, springt die Anwendung in die verlinkte Sicht und der Benutzer kann dort die Detailansicht aufrufen und die Daten bearbeiten.

Diese Links werden meist duch große, farbige Symbole markiert. Ein Beispiel: Durch das Klicken auf dieses Symbol   springt das System in die Sicht "Benutzer" und wählt die Person aus, deren Link geklickt wurde.

Weiterhin sind Links im System daran zu erkennen, dass sie die Farbe wechseln, wenn man mit dem Mauszeiger darüber fährt.
