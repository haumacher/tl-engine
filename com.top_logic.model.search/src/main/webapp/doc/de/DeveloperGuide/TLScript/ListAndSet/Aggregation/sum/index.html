Syntax

	sum($num1, ..., $numN)

$set.sum()

Beschreibung

Berechnet die Summe aus den gegebenen Werten. Alle Werte müssen dabei Zahlen sein.

Die Funktion kann entweder mit mehreren Argumenten oder mit einer Liste oder Menge als einzelnes Argument aufgerufen werden. null-Werte werden ausgefiltert und haben keinen Einfluss auf das Ergebnis.

Parameter

Name Typ Beschreibung Pflicht Default
num Zahl Die Zahlen, die miteinander addiert werden sollen. Einer von beiden muss definiert sein.
set Menge Eine Menge oder Liste mit Zahlen, die miteinander addiert werden sollen.

Rückgabewert

Typ: Zahl

Die Summe der angegebenen Elemente.

Beispiele

Einfache Addition

	sum(5, 10, 20)

Ausgabe: 35

Addition mit einer Menge

	list(1, 2, 3, 4).sum()

Ausgabe: 10

Addition mit Nullwerten

	list(1, null, 2, null, 3, 4).sum()

Ausgabe: 10

Die null-Werte werden herausgefiltert.
