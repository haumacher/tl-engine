Syntax

	$object.type()

Beschreibung

Liefert den Typ des Objektes object.

Parameter

Name Typ Beschreibung Pflicht Default
object Fachobjekt Das Objekt, dessen Typ ermittelte werden soll. ja

RÃ¼ckgabewert

Typ: Fachobjekt

Der Typ des Objekts object.

Beispiele

Einfach Typabfrage

	{
   object= new(`my.module:MyClass`);
   $object.type();
}

Ausgabe: my.module:MyType

Typenvergleich

	{
   object= new(`my.module:MyClass`);
   $object.type() == `my.module:MySuperClass`;
}

Ausgabe: false

Anders als instanceOf ergibt der Vergleich mit == nur true, wenn object genau dem Typ my.module:MyType und nicht wenn es nur ein Untertyp davon ist.
