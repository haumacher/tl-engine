Einer Sicht kann ein Dialog-Kommando hinzugefügt werden. Hierfür muss im Burger-Menü die Option "Dialog hinzufügen" gewählt werden.

Ein so definierter Dialog wird über einen Knopf geöffnet, der in der Sicht angezeigt wird, die den Dialog definiert. Der Knopf zum Öffnen des Dialogs wird zusammen mit dem Dialog definiert. Die verfügbaren Dialog-Templates werden im folgenden beschrieben.
