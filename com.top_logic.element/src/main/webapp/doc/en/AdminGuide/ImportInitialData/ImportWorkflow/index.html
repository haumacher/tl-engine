Abbildung 1:  Einspielen desWorkflows

  • Wechseln in den Reiter Prozesse : Prozessbrowser.
  • Dort das Kommando "Workflow hochladen ..." auswählen (vgl. Abbildung 1).
  • Die Datei "Homologation 1.0.bpml" auswählen.
  • "Hochladen" drücken.

Jetzt wird der aktuelle Workflow für die Anwendung installiert.

Der Import war erfolgreich, wenn der Dialog geschlossen wird und keine Fehlermeldungen angezeigt werden und die importierte Prozessdefinition angezeigt wird.
