Syntax

$input.translate(
    targetLang: $targetLang, 
    sourceLang: $sourceLang
)

Übersetzt den Eingabetext in die Zielsprache.

Parameter

Name Typ Beschreibung Pflicht Default
input Text Der zu übersetzende Text. ja
targetLang Zeichenkette Die Sprache in die übersetzt werden soll ja
sourceLang Zeichenkette Die Sprache der Eingabezeichenkette oder "auto" um die Sprache durch den Übersetzungsservice erkennen zu lassen. nein Die Sprache des angemeldeten Nutzers oder die Default-Sprache der Anwendung.

Rückgabewert

Typ: Text

Der Eingabewert in der geforderten Zielsprache.

Beispiele

Übersetzung von "Hello world!" nach Deutsch.

"Hello world!".translate(targetLang: "de")
