Wird in TopLogic die Eingabe eines Datums erwartet, kann dieses bei Drücken von ausgewählt werden. Es öffnet sich ein Kalenderdialog.

Die Steuerung der Datumsanzeige funktioniert wie folgt:

(1) - vorheriges Jahr

(2) - Vorheriger Monat

(3) - zum heutigen Datum springen

(4) - Jahresübersicht

(5) - nächster Monat

(6) - nächstes Jahr
