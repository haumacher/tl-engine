Syntax

	structType($key1, $key2, ..., $keyN)

Beschreibung

Ein Wörterbuch-Typ erzeugt Wörterbücher mit vordefinierten Schlüsseln. Über einen Wörterbuch-Typ können mehrere Wörterbücher mit denselben Schlüsseln aber unterschiedlichen Werten erzeugt werden (siehe Wörterbuch-Erzeugung).

Die folgenden Ausdrücke sind äquivalent:

structType("one", "two", "three")
 .newStruct(1, 2, 3)
{
  "one": 1,
  "two": 2,
  "three": 3
}

Parameter

Name Typ Beschreibung Pflicht Default
key Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Ein Schlüssel des Wörterbuchs. ja

Rückgabewert

Typ: Fachobjekt

Einen Wörterbuch-Typ, der zur Wörterbuch-Erzeugung weiterverwendet werden kann.

Beispiele

Mehrsprachiges Wörterbuch

	{
  eng = structType("Hello", "Goodbye");
  fr = $eng.newStruct("Bonjour", "Au revoir"); 
  ger = $eng.newStruct("Hallo", "Tschüss");
  $ger["Hello"];
}

Ausgabe: Hallo

Erzeugt Wörterbücher für die verschiedenen Übersetzunungen.
