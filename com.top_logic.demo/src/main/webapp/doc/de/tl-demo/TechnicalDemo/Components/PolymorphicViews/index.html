Dieser Tab demonstriert die Verwendung von Typ-Spezifischen Sichten. In der Objekte-Tabelle auf der linken Seite können Objekte unterschiedlicher Typen A, B und C angelegt werden. Auf der rechten Seite wird eine Sicht angezeigt, die sich je nach Objekttyp unterscheidet. Wird ein A-Objekt ausgewählt, erhält man genau ein Formular mit den Eigenschaften. Bei einem B-Objekt enthält die rechte Seite eine weitere Tab-Bar und bei einem C-Objekt wird die rechte Seite nocheinmal vertikal geteilt.
