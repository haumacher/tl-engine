Syntax

	$calendar.withDayAdded($value)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei der Tag um value erhöht wird. Die Addition des Wertes kann Auswirkungen auf andere Werte haben, zum Beispiel auf den Monat.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Tag geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
value Zahl Wert, der auf den aktuellen Tag des Kalenders addiert werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für den Tag.

Beispiele

	date(2012, 8, 10).toUserCalendar().withDayAdded(5).day()

Ausgabe: 15
