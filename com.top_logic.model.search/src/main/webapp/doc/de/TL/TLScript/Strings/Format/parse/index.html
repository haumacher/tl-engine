Syntax

	$format.parse($string)

Beschreibung

Verwendet das angegebene Format um den angegebenen Text zu parsen und gibt den so gelesenen Wert zurück.

Parameter

Name Typ Beschreibung Pflicht Default
format Fachobjekt Das Format, das verwendet werden soll (z.B. Zahlenformat, Datumsformat, Nachrichtenformat) ja
string Zeichenkette Die Zeichenkette, die geparst werden soll. ja

Rückgabewert

Typ: Fachobjekt

Die geparste Zeichenkette.

Beispiele

Datumsformat

	dateFormat("dd-MM-yy").parse("04-12-12")

Ausgabe: 04.12.2012

Gibt ein Datumsobjekt zurück. Siehe Datumsformat

Nachrichtenformat

	messageFormat("Good {0}!").parse("Good Morning!")

Ausgabe: Nachrichtenobjekt

Siehe Nachrichtenformat
