Syntax

	$calendar.second()

Beschreibung

Die Sekunde des gegebenen Kalenders.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Sekunde ermittelt werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja

Rückgabewert

Typ: Zahl

Eine Zahl, die die Sekunde des gegebenen Kalenders repräsentiert.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

	dateTime(2012, 8, 5, 8, 55, 23, 16).toUserCalendar().second()

Ausgabe: 23
