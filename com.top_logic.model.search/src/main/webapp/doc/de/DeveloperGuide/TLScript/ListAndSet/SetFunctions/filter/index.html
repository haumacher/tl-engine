Syntax

	$set.filter($func)

Beschreibung

Filtert set indem die Prädikatsfunktion func auf jedes seiner Elemente angewendet wird. In der resultierenden Menge sind nur die Elemente enthalten, welche die Prädikatsfunktion func als wahr bewertet hat.

Parameter

Name Typ Beschreibung Pflicht Default
set Menge Eine Menge, die gemäß func gefiltert werden soll. ja
func Funktion Prädikatsfunktion, die auf alle Elemente von set angewendet werden soll. ja

Rückgabewert

Typ: Menge

Eine neue Menge, die alle Elemente aus set beinhaltet, für die die Prädikatsfunktion true ergab.

Beispiele

Zahlenfilter

	list(1, 3, 5, 3, 7, 8, 3, 9)
  .filter(x -> $x > 4)

Ausgabe: Eine Menge mit den Elementen [5, 7, 8, 9].

Für alle anderen Zahlen gibt die Funktion false zurück.
