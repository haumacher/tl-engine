Syntax

	toMillis($date)

Beschreibung

Konstruiert einen Millisekundenwert (seit 1970-01-01:00:00:00.000) von dem gegebenen Datumswert date.

Die Umkehrfunktion ist toDate($number).

Parameter

Name Typ Beschreibung Pflicht Default
date Fachobjekt Ein Datumswert, der in Millisekunden umgewandelt werden soll ja

Rückgabewert

Typ: Zahl

Die Millisekunden, die den Datumswert seit 1970-01-01:00:00:00.000 repräsentieren.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

	date(2012, 5, 5).toMillis()

Ausgabe: 1.338.822.000.000
