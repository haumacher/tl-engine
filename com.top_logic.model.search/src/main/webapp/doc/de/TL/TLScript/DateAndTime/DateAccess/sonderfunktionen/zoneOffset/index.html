Syntax

	$calendar.zoneOffset()

Beschreibung

Der UTC-Offset des gegebenen Kalenders in Millisekunden (siehe toSystemCalendar und toUserCalendar).

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen UTC-Offset ermittelt werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja

Rückgabewert

Typ: Zahl

Eine Zahl, die den UTC-Offset set des gegebenen Kalenders in Millisekunden angibt.

Beispiele

	date(2012, 8, 15).toUserCalendar().zoneOffset()

Ausgabe für die Zeitzone Arizona (UTC -7): -25.200.000

Umgerechnet in Stunden ergibt das -7.
