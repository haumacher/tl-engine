Die Sicht stellt ein Demo für Auswahlkomponenten in verschiedenen Konfigurationen dar. Um Daten für die Tests zu haben, müssen diese im Hauptreiter "Strukturen" angelegt werden.

Die erste Auswahl ist eine Mehrfachauswahl eines Kontextes (Objekte vom Typ A). Alle Kinder der ausgewählten Strukturknoten werden als Optionen in der zweiten Auswahl "Kinder" (Einfachselektion) angeboten. Das dort ausgewählte Objekt wird im nebenliegenden Formular angezeigt.

Die dritte Auswahl "Standard-Mehrfachauswahl" hat als Optionen auch alle Kinder der Kontext-Auswahl, belegt die Auwahl aber mit allen zur Auswahl stehenden Objekten vom Typ C vor. Eine Änderung der Auswahl hat keinen weiteren Effekt.

Die Auswahl "Potentielle Leerselektion" hat dieselben Optionen, aber selektiert immer die Elemente vor, welche eine "1" im Namen haben. Damit kann im Standardsetup (z.B. bei Kontextwahl "A2") auch eine leere Defaultselektion erreicht werden.
