Syntax

	average($num1, ..., $numN)

$set.average()

Beschreibung

Berechnet den Durchschnittswert aller gegebenen Werte. Alle Werte müssen Zahlen sein. Die Funktion kann entweder mit mehreren Argumenten oder mit einer Menge bzw. Liste von Zahlen als Argument aufgerufen werden. null-Werte werden ausgefiltert und haben keinen Einfluss auf das Ergebnis.

Parameter

Name Typ Beschreibung Pflicht Default
number Zahl Eine Zahl, die in die Berechnung des Durchschnitts miteinfließen soll. Einer von beiden muss definiert sein.
set Menge Eine Menge von Zahlen, die in die Berechnung des Durchschnitts miteinfließen sollen.

Rückgabewert

Typ: Zahl

Der Durchschnitt der angegebenen Zahlen.

Beispiele

Einfacher Durchschnitt

	average(1, 2, 3, 4)

Ausgabe: 2,5

Durchschnitt mit einer Liste

	list(1, 2, 3, 4).average()

Ausgabe: 2,5

Durchschnitt mit null

	list(1, 2, 3, 4, null).average()

Ausgabe: 2,5
