Kombination von horizontalen und vertikalen Trennern

Möchte man ein komplexeres Layout aus einer Verschtelung von nebeneinander und untereinander angeordneten Komponenten erzeugen, so beginnt man mit der Hauptrichtung und teilt die Sicht z.B. zuerst horiontal. Anschließend füllt man einen oder mehrere Platzhalter mit einem neuen Layout. Dieses innere Layout kann dann eine andere Richtung erhalten.
