Syntax

	$calendar.withDay(day)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei der Tag auf day gesetzt wird. Durch das Verändern des Tages können sich auch andere Werte wie zum Beispiel die Kalenderwoche ändern.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Tag geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
day Zahl Wert für den neuen Tag, der für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für den Tag.

Beispiele

	date(2012, 8, 15).toUserCalendar().withDay(22).day()

Ausgabe: 22
