Voraussetzung für die ordnungsgemäße Lauffähigkeit von TopLogic ist eine gültige Lizenz. In dieser werden der Gültigkeitszeitraum und die Anzahl Benutzer festgehalten.

Damit eine gültige Lizenz in TopLogic vorhanden ist, muss ein Lizenzschlüssel eingespielt werden. Der Lizenzschlüssel wird in einer eigenen TopLogic-App verwaltet.

Über den Reiter Administration : Monitor : Monitor gelangt man zur Lizenzverwaltung.

Abbildung 1: Einspielen eines Lizenzschlüssels

Nach erfolgreicher Einspielung des Lizenzschlüssels werden die Lizenz-Informationen im Monitor angezeigt.
