Syntax

	today()

Beschreibung

Konstruiert einen Datumswert, der den aktuellen Tag des Jahres aus Sicht des Benutzerkalenders darstellt, der durch einen tl.core:Date Wert (im Systemkalender) repräsentiert wird.

Rückgabewert

Typ: Fachobjekt

Ein Datumswert, der den atuellen Tag gemäß des Benutzerkalenders repräsentiert.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

Unformatierts Datum

	today()

Ausgabe: 06.10.2021

Formatiertes Datum

	dateFormat("dd-MM-yyy hh:mm:ss").format(today())

Ausgabe: 06-10-2021 05:00:00
