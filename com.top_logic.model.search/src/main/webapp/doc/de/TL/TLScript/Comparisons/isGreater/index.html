Syntax

	$e1.isGreater($e2)

Beschreibung

Boolescher Ausdruck, der zu "wahr" ausgewertet wird, wenn das Ergebnis der Auswertung von e1 größer ist als das Ergebnis der Auswertung von e2. Siehe Größer als für alternative Funktionen.

Parameter

Name Typ Beschreibung Pflicht Default
e1 Zahl/Zeichenkette/Menge Objekt, das mit e2 verglichen werden soll. ja
e2 Zahl/Zeichenkette/Menge Objekt, das mit e1 verglichen werden soll. ja

Rückgabewert

Typ: Wahrheitswert

true, wenn das Ergebnis der Auswertung besagt, dass e1 größer als e2 ist.

Beispiele

Zahlen

	2.isGreater(1)

Ausgabe: true

Ungleiche Mengen

	list(1, 2, 4).isGreater(list(1, 2, 3))

Ausgabe: true

Da Mengen elementweise miteinander verglichen werden, ist die erste Menge größer als die zweite, denn die Zahl 4 ist größer als die Zahl 3 (siehe Vergleichsoperatoren).

Gleiche Mengen

	list(1, 2, 3).isGreater(list(1, 2, 3))

Ausgabe: false

Die Mengen sind gleich groß.
