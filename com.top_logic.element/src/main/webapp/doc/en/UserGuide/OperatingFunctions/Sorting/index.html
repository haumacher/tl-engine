Zentraler Bestandteil in TopLogic ist die Anzeige der Daten in einer Tabelle. Um das angezeigte Tabellenergebnis besser bearbeitbar zu machen kann grundsätzlich nach jeder Tabellenspalte sortiert werden. Dazu gibt es die Möglichkeit im Spaltenkopf die Tabelle auf- und absteigend zu sortieren . Wurde eine Tabelle nach einer Spalte sortiert wird aus diesem Symbol dieses hier .

Um eine Sortierung über mehrere Spalten durchzuführen, wird über das Burger-Menü (1) der Unterpunkt Sortierung ausgewählt. Im angezeigten Popup-Window kann dann die Tabelle über jede Spalte sortiert werden (2).
