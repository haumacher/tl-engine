Syntax

	$set1.intersection($set2)

Beschreibung

Berechnet den Schnitt der Mengen set1 und set2.

Parameter

Name Typ Beschreibung Pflicht Default
set1 Menge Eine Menge, für die überprüft werden soll, ob und welche Elemente auch in set2 enthalten sind. ja
set2 Menge Eine Menge, für die überprüft werden soll, ob und welche Elemente auch in set1 enthalten sind. ja

Rückgabewert

Typ: Menge

Eine Menge, die alle Elemente enthält, die sowohl in set1, als auch in set2 enthalten sind.

Beispiele

Schnittmenge

	list(1, 2, 3).intersection(list(2, 3))

Ausgabe: Eine Menge mit den Elementen [2, 3].

Schnittmenge bei doppelten Werten

	list(1, 2, 2, 3).intersection(list(2, 2, 3))

Ausgabe: Eine Menge mit den Elementen [2, 3].

Doppelte Werte tauchen nicht doppelt im Ergebnis auf.

Keine Schnittmenge

	list(1, 2, 3).intersection(list(5, 6))

Ausgabe: Eine leere Menge
