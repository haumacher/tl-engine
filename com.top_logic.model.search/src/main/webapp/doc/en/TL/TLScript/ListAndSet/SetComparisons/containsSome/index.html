Syntax

	$set1.containsSome($set2)

Description

Checks whether set1 contains at least one element of the set2.

Parameters

Name Type Description Mandatory Default
set1 Set A set for which it is to be checked whether at least one element from set2 is contained. yes
set2 Set A set for which to check if at least one element is contained in set1. yes

Return value

Type: Boolean

true, if at least one element from set2 is contained in set1.

Examples

Set contains element

	list(1, 2, 3, 4).containsSome(list(1, 2))

Output: true

Set contains none of the elements

	list(1, 2, 3, 4).containsSome(list(5, 6, 7))

Output: false
