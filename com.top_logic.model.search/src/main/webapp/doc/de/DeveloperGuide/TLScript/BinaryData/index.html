Unter Binärdaten versteht man alle unstrukturierten Daten wie Dokumente, Word-Dateien, PowerPoint-Folien, Text-Dateien, die mit einer TopLogic-Anwendung verwaltet werden. Binärdaten können in Attributen vom Typ "Binär" (tl.core:Binary) gespeichert werden. Ein solcher binärer Wert besteht immer aus einem Namen, einem Inhaltstyp ("MIME-Type") und dem eigentlichen Datenstrom. TL-Script bietet Unterstützung für den Umgang und die Erzeugung solcher binärer Werte über eine Reihe von Funktionen.
