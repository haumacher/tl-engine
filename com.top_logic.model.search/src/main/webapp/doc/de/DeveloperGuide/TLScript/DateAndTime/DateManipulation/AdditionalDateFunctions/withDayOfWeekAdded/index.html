Syntax

	$calendar.withDayOfWeekAdded($value)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei der Tag der Woche um value erhöht wird. Die Addition des Wertes kann Auswirkungen auf andere Werte haben, zum Beispiel auf den Tag.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Tag der Woche geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
value Zahl Wert um den der Tag der Woche erhöht werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für den Tag der Woche.

Beispiele

DayOfWeek nach Anpassung

	dateTime(2012, 8, 14, 7, 55, 24, 13).toUserCalendar().withDayOfWeekAdded(1).dayOfWeek()

Ausgabe: 7

Tag nach Anpassung

	dateTime(2012, 8, 14, 7, 55, 24, 13).toUserCalendar().withDayOfWeekAdded(1).day()

Ausgabe: 15

Die Anpassung von dayOfWeek ändert auch andere Werte wie den Tag.
