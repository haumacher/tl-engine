Syntax

	icon($iconSource)

Beschreibung

Erstellt ein ThemeImage mit gegebener Quelle.

Parameter

Name Typ Beschreibung Pflicht Default
iconSource Zeichenkette Quelle aus der das Bild geladen wird ja

Rückgabewert

Typ: ThemeImage

Beispiele

	icon("css:far fa-thumbs-up")

icon("/images/img.png")
