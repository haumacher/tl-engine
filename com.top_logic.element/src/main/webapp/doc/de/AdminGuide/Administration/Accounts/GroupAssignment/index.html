Hier können die Gruppenzuordnungen der einzelnen Benutzerkonten bearbeitet werden. D.h. einzelne Benutzerkonten werden fachlichen Benutzergruppen wie z.B. „Stammdatenpfleger“ oder „Systemadministrator“ zugeordnet. Die Mitglieder dieser Gruppen erhalten korrespondierende Rollen mit den dafür festgelegten Berechtigungen. Ein neu angelegter Benutzer wird per Default der Gruppe "Benutzer" zugeordnet.

Dies bedeutet, dass Berechtigungen nur über Rollen und Gruppen den Benutzern zugeordnet werden. 

Im Anzeigemodus sieht man nur die zugeordneten Gruppen. Im Bearbeitungsmodus werden zusätzlich alle noch nicht zugeordneten, aber auswählbaren Gruppen angezeigt.
