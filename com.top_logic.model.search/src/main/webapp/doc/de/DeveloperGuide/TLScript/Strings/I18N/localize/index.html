Syntax

$input.localize(lang: $language)

Wertet einen internationalisierten Text in einer gegeben Sprache aus.

Parameter

Name Typ Beschreibung Pflicht Default
input Internationalisierter Text Der internationalisierte Text dessen Lokalisierung bestimmt werden soll. ja
language Zeichenkette Der Code der Sprache zu der die Lokalisierung vorgenommen werden. nein Die Sprache des angemeldeten Nutzers oder die Default-Sprache der Anwendung.

Rückgabewert

Typ: Text

Der Wert den ein Nutzer mit der ausgewählten Sprache sehen wird.

Beispiele

{
  resKey = {"de": "Hallo", "en": "Hello"}.internationalize();

  $resKey.localize("en"); // Hello
  $resKey.localize("de"); // Hallo
}
